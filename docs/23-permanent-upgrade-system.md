# 저승 수련 & 메타 진행 시스템

## 1. 개요

### 핵심 루프
```
[런 플레이] → [런 종료 (클리어 or 사망)]
     │                    │
     │              [넋 획득]
     │                    │
     │              [저승 수련 투자]
     │                    │
     └────── [다음 런 (더 강해진 상태)] ←──┘
```

### 넋 (Soul) — 메타 통화
- 매 런에서 획득하는 **영구 자원**
- 런 실패해도 획득량의 70% 유지
- 엽전과 별개 (엽전은 런 내 소모품, 넋은 메타)

| 획득 방법 | 넋 |
|----------|-----|
| 보스 격파 (관문 1~5) | 10~30 |
| 보스 격파 (관문 6~10) | 30~80 |
| 무한 모드 웨이브 5 | 15 |
| 무한 모드 웨이브 10 | 30 |
| 무한 모드 웨이브 25 | 100 |
| 무한 모드 웨이브 50 | 300 |
| 업적 달성 | 20~200 |
| Go 3회 성공 | 50 |
| 런 최초 클리어 | 500 (1회) |

## 2. 저승 수련 트리 (Permanent Upgrade Tree)

### 트리 구조
```
                    [넋의 제단]
                         │
        ┌────────────────┼────────────────┐
        │                │                │
   [패의 길]        [부적의 길]       [생존의 길]
   카드/점수 강화    부적 시스템 강화    체력/방어 강화
        │                │                │
   ┌────┼────┐      ┌────┼────┐      ┌────┼────┐
   │    │    │      │    │    │      │    │    │
  칩   배수  덱     슬롯  발동  합성   목숨  Go   회복
```

### [패의 길] — 카드/데미지 저승 수련

| 강화 | 레벨 | 효과 | 비용 |
|------|------|------|------|
| 기본 칩 증가 | 1~10 | 모든 족보 칩 +5/레벨 | 20/40/60/.../200 |
| 기본 배수 증가 | 1~5 | 기본 배수 +1/레벨 | 50/100/200/400/800 |
| 시작 손패 | 1~3 | 시작 손패 +1/레벨 | 100/300/600 |
| 덱 압축 | 1~4 | 덱에서 피 카드 2장 제거/레벨 | 80/160/320/640 |
| 더블 매칭 | 1 | SingleMatch 시 30% 확률 추가 드로우 | 500 |
| 족보 보너스 | 1~5 | 첫 족보 완성 시 보너스 칩 +20/레벨 | 60/120/240/480/960 |
| 쓸 보너스 | 1~3 | 쓸(TripleMatch) 시 배수 +1/레벨 | 100/200/400 |

**최대 투자: 약 8,000 넋**

### [부적의 길] — 부적 시스템 저승 수련

| 강화 | 레벨 | 효과 | 비용 |
|------|------|------|------|
| 부적 슬롯 확장 | 1~3 | 최대 부적 수 +1/레벨 (5→8) | 200/500/1000 |
| 부적 발동률 | 1~5 | 확률 부적 발동률 +5%/레벨 | 40/80/160/320/640 |
| 장터 부적 수 | 1~2 | 저승 장터 부적 진열 +1/레벨 | 150/400 |
| 전설 등장률 | 1~3 | 저승 장터 전설 부적 등장 +5%/레벨 | 300/600/1200 |
| 부적 합성 해금 | 1 | 같은 등급 부적 2개 → 상위 등급 1개 | 800 |
| 시작 부적 슬롯 | 1~2 | 런 시작 시 선택 부적 +1/레벨 | 500/1000 |
| 저주 저항 | 1~3 | 저주 부적 효과 -20%/레벨 | 100/200/400 |

**최대 투자: 약 8,500 넋**

### [생존의 길] — 체력/방어 저승 수련

| 강화 | 레벨 | 효과 | 비용 |
|------|------|------|------|
| 최대 목숨 | 1~3 | 시작 목숨 +1/레벨 (3→6) | 150/300/600 |
| Go 보험 | 1~2 | Go 실패 시 30%/60% 확률로 면제 | 300/800 |
| 시작 엽전 | 1~5 | 시작 엽전 +30/레벨 | 30/60/120/240/480 |
| 장터 할인 | 1~3 | 저승 장터 가격 -10%/레벨 | 100/250/500 |
| 이벤트 보너스 | 1~3 | 이벤트 보상 +20%/레벨 | 80/160/320 |
| 부활 | 1 | 런 당 1회 즉사 면제 | 1500 |
| 보스 HP 감소 | 1~5 | 모든 보스 HP -3%/레벨 | 100/200/400/800/1600 |

**최대 투자: 약 8,600 넋**

### 전체 저승 수련 비용 합계
```
패의 길:  ~8,000
부적의 길: ~8,500
생존의 길: ~8,600
─────────────────
합계: ~25,100 넋

평균 런(관문 5 클리어): ~200 넋
→ 전체 맥스 수련: ~125런 (약 50~80시간)
→ 핵심 수련(70%): ~40런 (약 20~30시간)
→ 적절한 장기 목표 제공
```

## 4. 뉴게임+ (윤회 시스템)

### 컨셉
스토리 클리어 후 **윤회의 문**을 통해 새 런 시작.
이전 런의 일부가 다음 런에 영향.

### 윤회 단계
| 윤회 | 해금 조건 | 변경 사항 | 보너스 |
|------|---------|----------|--------|
| 1차 윤회 | 스토리 클리어 | 보스 기믹 강화 | 시작 부적 2개 선택 |
| 2차 윤회 | 1차 윤회 클리어 | 새 기믹 추가 | 시작 카드 강화 3장 |
| 3차 윤회 | 2차 윤회 클리어 | 보스 2페이즈 추가 | 시작 엽전 2배 |
| 무한 윤회 | 3차 윤회 클리어 | 매 윤회 +10% 난이도 | 윤회 횟수 표시 |

### 윤회 캐리오버 (이월 항목)
| 항목 | 이월 여부 | 비고 |
|------|---------|------|
| 저승 수련 | ✅ 유지 | 넋 투자분 |
| 업적 | ✅ 유지 | 영구 |
| 도감 | ✅ 유지 | 영구 |
| 길들인 도깨비 | ✅ 유지 | 영구 버프 |
| 카드 강화 등급 | ⚠️ 1단계 하락 | 전설→신통 |
| 변이 | ❌ 초기화 | 런 내 한정 |
| 부적 | ❌ 초기화 | 런 내 한정 |
| 엽전 | ❌ 초기화 | 기본값 시작 |
| 목숨 | ❌ 초기화 | 기본값 시작 |

## 5. 시즌 패스 (출시 후 — 무료)

### 시즌 구조
- 3개월 주기 시즌 (무료, 유료 DLC 아님)
- 시즌별 테마 + 신규 콘텐츠
- 시즌 미션 완료 → 보상 (스킨, 넋, 부적)

| 시즌 | 테마 | 신규 콘텐츠 |
|------|------|-----------|
| S1: 각성 | 기본 | 출시 콘텐츠 |
| S2: 귀환 | 봄 | 새 부적 5종, 이벤트 3종, 카드 스킨 |
| S3: 황천 | 여름 | 무한 모드 재앙 보스 2종, 부적 5종 |
| S4: 윤회 | 가을 | 뉴게임++ 모드, 도깨비 2종 |
