## 공통 디자인 시스템 (모든 프롬프트 앞에 붙이기)

```
[공통 스타일]
- 게임: "도깨비의 패" — 한국 저승 세계관 화투 로그라이트
- 해상도: 1280×720
- 아트: 다크 판타지 + 한국 전통 미학
- 배경: #090916 (매우 어두운 남색)
- 패널: #110E21 반투명(94%), 둥근 모서리 6px, box-shadow 3px
- 테두리: #382861 (은은한 보라), 1px
- 제목 텍스트: #FFD100 (금색), Bold, text-shadow 글로우
- 본문 텍스트: #FFFFFF
- 보조 텍스트: #6B6B80
- 위험/경고: #D12720 (붉은)
- 안전/긍정: #1A8C33 (녹색)
- 시너지/특수: #40E6D9 (시안)
- 전설등급: #8033A6 (보라)
- 버튼: 둥근 6px, 그림자 2px, 상단 미세 광택, 호버 시 밝아짐 15%
- 폰트: Pretendard (한국어), 제목 Bold, 본문 Regular
- 레이아웃: 모든 요소 화면 중앙 정렬
- 분위기: 어둡고 고요한 저승, 도깨비불이 은은히 떠다님, 먹물 질감
- 화면 가장자리: 비네트 (가장자리 어두움)
```

```

   중: "♥♥♥♥♥♡♡♡♡♡" 체력 (빨간 하트, 빈 하트)
   우: "50냥" 엽전 (금색)

2. 보스 영역 (중앙 정렬, 480×145px):
   패널 배경 + 그림자

   좌측: 보스 아이콘 (105×105px)
     - 어두운 붉은 사각형 + "鬼" 큰 글자 (붉은)
     - 하단: "Lv.3"

   우측:
     - 보스 이름: "먹보 도깨비" (금색, Bold 20pt)
     - 기믹: "2턴마다 최고 패 1장 먹기" (회색)
     - HP 바 (넓게, 높이 16px):
       - 배경 #230D0D
       - 채우기: >50% #C72419 / 30~50% #E07A19 / <30% #FF3333
       - 상단 광택 (흰색 15% 반투명)
       - 텍스트: "HP 75 / 100" 중앙
     - "판 2/6 · 시너지 1/5 · 고 0회" (회색)
     - 등록된 시너지: "[홍단] [알리]" (금색 뱃지)

3. 시너지 점수 (중앙 정렬, 320×38px 패널):
   "칩 120" (흰색) "× 2.5" (시안) "= 300" (금색, Bold)

4. 손패 영역 (화면 중앙):
   상단: "시너지 카드: 2/5장 선택" (금색)
   카드 10장 가로 정렬 (간격 6px, 중앙 정렬)

   각 카드 90×130px:
     - 상단 헤더 (28px): 월 색상
       광: 금색 / 띠-홍단: 붉은 / 띠-청단: 파란 / 띠-초단: 녹색
       열끗: 하늘색 / 피: 회색
     - 헤더 텍스트: "3월" (검정)
     - 중앙: 타입 아이콘 (★/━/◆/·)
     - 하단: 카드 이름 "3월 광(막)"

   선택된 카드: 위로 20px 올라감 + 금색 테두리 2px
   호버 카드: 위로 6px + 약간 밝아짐

5. 부적 슬롯 (카드 위 좌측):
   "부적:" + "[피의 맹세] [도깨비 방망이]" (시안)

6. 고정 버튼바 (최하단 50px):
   배경: #100A23 (95% 불투명)
   상단 구분선: #382861

   [시너지 등록 모드]: "시너지 등록!" 붉은 버튼 중앙
   [고/스톱 모드]: "고!" 붉은 + "스톱" 파란 + 리스크 텍스트
   [공격 모드]: "공격!" 붉은 + 섯다 미리보기 텍스트

메시지 로그 (좌측, 시너지 아래):
   최근 5줄, 위가 최신, 아래로 갈수록 투명해짐
   금색 텍스트
```
