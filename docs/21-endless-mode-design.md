# 무한 진행 시스템 — 무한 윤회 (Infinite Cycle)

> **NOTE:** 이 문서는 원래 "별도의 무한 모드"로 설계되었으나,
> 현재 게임 구조 변경에 따라 **메인 게임 자체가 무한 진행**이 되었다.
> 아래 내용은 더 이상 별도 모드가 아닌, **코어 게임 루프의 일부**로 통합되었다.
> 무한 윤회 구조의 기본 설계는 `15-roguelike-progression.md` 참조.
> 윤회 스케일링 수치는 `10-balance-design.md` 참조.
>
> **핵심 컨셉:** 화투패를 모아 족보(고스톱+섯다+저승 고유)를 만들고, 그 족보로 보스 HP를 깎아 물리치는 전투.

## 1. 컨셉

### 핵심 아이디어
> 게임에 최종 스테이지는 없다. 메인 게임 자체가 **무한 윤회(Infinite Cycle)** 구조.
> 로그라이크 + Vampire Survivors 스타일의 "끝없는 도전 + 점진적 강화"를
> 화투 카드 게임 + 보스 HP 전투에 적용.
> 10개 관문(Gate)으로 구성된 윤회가 난이도를 높여가며 무한 반복된다.
> 매 관문마다 랜덤 보스가 등장하고, 족보 데미지로 보스 HP를 깎아 물리친다.

### 스토리 맥락
```
저승에 끌려온 망자가 도깨비들과 화투판을 벌이지만,
이승으로 가는 길은 끝이 없는 윤회였다.

관문마다 나타나는 도깨비의 넋을 꺾어도,
또 다른 관문이 기다리고 있다.

"윤회란 끝이 없는 법이지.
 하지만 네 패가 강하다면... 길은 열릴 것이다."

— 윤회는 계속된다.
```

## 2. 무한 윤회 구조 (코어 게임 루프)

### 윤회 구조
```
[윤회 1] → [윤회 2] → [윤회 3] → ... → [∞]
     │            │            │
  10개 관문    10개 관문    10개 관문
     │            │            │
  매 관문 = 랜덤 보스 (HP 전투)

각 관문: 랜덤 보스 등장 → 족보 데미지로 HP 깎기 → 보스 반격 → HP 0 = 승리
관문 클리어 시: 강화 선택 (3택 1) + 저승장터
```

### 전투 흐름 (관문 1개)
```
[패 분배] → [턴: 매칭으로 카드 수집]
    → [족보 완성 (고스톱+섯다+저승)] → [Go/Stop 선택]
    → [Stop: 데미지 정산] → [보스 HP 감소]
    → [보스 반격 (목숨/손패 감소)]
    → [보스 HP > 0? → 다음 라운드 / HP = 0 → 승리!]
    → [격노 페이즈 (HP 50%↓): 기믹 가속 + 반격 강화]
    → [필살 페이즈 (HP 25%↓): 기믹 매 턴 + 반격 2배]
```

### 윤회별 구조
| 윤회 구간 | 관문 난이도 | 보스 HP 범위 | 기믹 수 | 비고 |
|-----------|-----------|-------------|---------|------|
| 윤회 1 | 학습 | 300~3000 | 1 | 랜덤 보스 (고정 순서 없음) |
| 윤회 2~3 | 도전 | 390~4800 | 1~2 | 격노 페이즈 체감 시작 |
| 윤회 4~7 | 고급 | 660~8400 | 2~3 | 섯다 족보 필수, 파츠 복합 기믹 |
| 윤회 8~15 | 극한 | 1110~16500 | 3~4 | 전설 파츠, 3종 족보 극대화 필수 |
| 윤회 16+ | 무한 | 스케일링 계속 | 4 | 리더보드 경쟁 영역 |

### 난이도 스케일링
```
윤회 계수 = 1 + (윤회번호 - 1) × 0.3
보스 HP = 관문번호 × 300 × 윤회 계수

윤회 1, 관문 1:  300 HP
윤회 1, 관문 10: 3000 HP
윤회 3, 관문 10: 4800 HP
윤회 10, 관문 10: ~11100 HP
윤회 20, 관문 10: ~18600 HP
→ 3종 족보 시너지 + 부적 + 강화 누적 없이는 윤회 10+ 불가능
```

## 3. 관문 클리어 강화 시스템

### 관문 클리어 보상: 강화 선택 (3개 중 1개)
Vampire Survivors처럼 매 관문 보스 격파 후 **3개의 강화 옵션** 중 택 1.
이는 메인 게임의 코어 루프에 포함된 핵심 시스템이다.

#### 강화 카테고리

**A. 패 강화 (Card Power)**
| 강화 | 효과 | 등급 |
|------|------|------|
| 칩 강화 I~V | 모든 족보 칩 +10/20/30/50/100 | 일반~전설 |
| 배수 강화 I~V | 모든 족보 배수 +1/2/3/5/10 | 일반~전설 |
| 피 가치 증가 | 피 패 기본점수 +1 (영구) | 일반 |
| 광 초월 | 광 패 칩 +50, 배수 +3 | 전설 |
| 띠 강화 | 띠 족보 칩 2배 | 희귀 |
| 만능패 | 랜덤 패 1장이 와일드카드 (아무 월 매칭) | 전설 |

**B. 부적 강화 (Talisman Power)**
| 강화 | 효과 | 등급 |
|------|------|------|
| 부적 슬롯 +1 | 최대 부적 수 증가 | 희귀 |
| 부적 진화 | 보유 부적 1개를 상위 등급으로 | 전설 |
| 이중 발동 | 부적 효과 2회 적용 (1개 선택) | 전설 |
| 부적 자유 | 저주 부적 1개 제거 가능 | 희귀 |
| 부적 연금술 | 부적 2개 합성 → 전설 1개 | 전설 |

**C. 생존 강화 (Survival)**
| 강화 | 효과 | 등급 |
|------|------|------|
| 목숨 +1 | 최대 목숨 증가 (캡 없음) | 희귀 |
| 손패 +1 | 시작 손패 수 영구 증가 | 일반 |
| 바닥패 -1 | 바닥패 감소 (매칭 확률 증가) | 일반 |
| Go 보험 | Go 실패 시 목숨 소모 대신 점수 0 | 전설 |
| 부활 | 사망 시 1회 부활 (웨이브 유지) | 전설 |

**D. 특수 강화 (Special)**
| 강화 | 효과 | 등급 |
|------|------|------|
| 시간 역행 | 턴 1회 되감기 가능 | 전설 |
| 도깨비 동맹 | 이전 보스 기믹을 자신에게 적용 (유리하게) | 희귀 |
| 황천의 축복 | 다음 5관문 보스 HP -30% | 희귀 |
| 업보 소멸 | 현재 적용 중인 기믹 1개 영구 제거 | 전설 |
| 윤회 단편 | 족보 판정 시 카드 1장을 다른 월로 취급 | 전설 |

### 강화 등장 확률
| 등급 | 확률 | 웨이브 25+ | 웨이브 50+ |
|------|------|-----------|-----------|
| 일반 | 60% | 40% | 20% |
| 희귀 | 30% | 35% | 35% |
| 전설 | 10% | 25% | 45% |

## 4. 재앙 보스 (Calamity Boss)

### 특정 윤회 도달 시 등장하는 특수 보스
기존 보스의 변형이 아닌, **윤회 고유의 초월 존재**. 일반 보스 대신 등장. 높은 HP + 고유 기믹.

#### 재앙 보스 목록

**재앙 1: 백골대장(白骨大將) — 윤회 3 보스**
```
"황천의 뼈로 만들어진 거인"

HP: 5000
기믹: '백골화'
- 매 턴 종료 시 획득 카드 중 랜덤 1장이 '해골 패'로 변함
- 해골 패: 어떤 월에도 속하지 않음, 족보/섯다 무효
- 해골 패 3장 이상 모이면 즉사
- 대응: 해골 패를 바닥에 내려 제거 가능 (1턴 소모)
- 격노(HP 50%): 해골 패 변환이 매 턴 2장으로 증가
- 반격: 목숨 -1 + 랜덤 부적 비활성화 1턴

보상: 전설 강화 + '해골왕의 뿔' 특수 부적
```

**재앙 2: 구미호 왕(九尾狐王) — 윤회 5 보스**
```
"아홉 꼬리의 환영이 진실을 삼킨다"

HP: 10000
기믹: '구미 환영'
- 바닥패, 손패, 뽑기패 모든 것에 환영이 섞임
- 환영 비율: 30% (전체 카드의 30%가 가짜)
- 환영을 매칭하면 데미지 -50 칩
- 진짜 카드에 미세한 여우불(초록빛) 힌트
- + 3턴마다 손패 전체 셔플
- 격노(HP 50%): 환영 비율 50%로 증가
- 반격: 목숨 -1 + 부적 1개 환영으로 교체 (1라운드)

보상: 전설 강화 2개 + '여우구슬' 특수 부적
```

**재앙 3: 이무기(螭龍) — 윤회 8 보스**
```
"천년을 기다렸으나 용이 되지 못한 존재"

HP: 20000
기믹: '승천 경쟁'
- 이무기도 자체적으로 HP를 회복함 (매 턴 +200 HP)
- 플레이어가 회복량보다 큰 데미지를 줘야 HP가 줄어듦
- 이무기의 HP 회복이 실시간으로 표시됨
- + Go 선택 시 이무기도 회복량 2배 (배수 경쟁)
- 격노(HP 50%): 매 턴 회복량 +400으로 증가
- 반격: 목숨 -2 + 손패 -2장

보상: 전설 강화 3개 + '용의 여의주' 특수 부적
```

**재앙 4: 저승꽃 (피안화의 왕) — 윤회 10 보스**
```
"기억과 망각의 경계에서 피어난 존재"

HP: 30000
기믹: '망각'
- 매 라운드 시작 시 플레이어의 강화 1개가 일시 비활성화
- 5턴마다 부적 전체 셔플 (다른 부적으로 교체)
- 해당 라운드에서만 적용, 다음 라운드에 복구
- + 족보 이름이 표시되지 않음 (기억해야 함)
- 격노(HP 50%): 강화 비활성화 2개 + 부적 셔플 3턴마다
- 반격: 목숨 -2 + 강화 1개 영구 제거 (!)

보상: '초월' 등급 강화 + 시크릿 언락
```

## 5. 리더보드 & 경쟁

### Steam 리더보드
| 보드 | 기준 | 리셋 |
|------|------|------|
| 최고 웨이브 | 도달 웨이브 수 | 영구 |
| 최고 점수 | 단일 라운드 최고 점수 | 영구 |

### 주간 도전 (Weekly Challenge)
- 매주 동일 시드 + 특수 규칙
- 전 유저가 같은 조건으로 경쟁

| 주차 예시 | 특수 규칙 |
|----------|----------|
| 광의 주 | 광 패 2배 등장, 광 점수 3배 |
| 저주의 주 | 시작 시 저주 부적 3개 장착 |
| Go의 주 | Go 배수 2배, 실패 패널티 없음 |
| 가난의 주 | 상점 없음, 강화만 가능 |

## 7. Vampire Survivors DNA 적용 포인트 (코어 게임 루프)

> 이 요소들은 별도 모드가 아닌 **메인 게임의 핵심 시스템**으로 통합되었다.

| VS 요소 | 도깨비의 패 적용 |
|---------|---------------|
| 무한 몹 웨이브 | 무한 윤회 (10 관문 × 무한 반복) |
| 레벨업 시 스킬 선택 | 관문 클리어 시 강화 선택 (3택 1) |
| 무기 진화 | 부적 진화 (합성/승급) |
| 시간 경과 난이도 | 윤회 회차별 스케일링 (x0.3/윤회) + 보스 HP 증가 |
| 보스 웨이브 | 매 관문 랜덤 보스 (HP 전투) + 재앙 보스 (특정 윤회) |
| 골드 → 영구 업그레이드 | 엽전(냥) → 런 내 / 넋(Soul) → 영구 강화 |
| 캐릭터 선택 | 시작 부적 선택 + 동료 도깨비 |
| 리더보드 | Steam 리더보드 + 주간 도전 |
| 선택적 종료 | 윤회 끝에서 계속 or 종료 선택 |
