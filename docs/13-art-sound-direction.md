## 2. 화투패 아트

### 기본 스타일
- **전통 화투 디자인을 저승 테마로 리디자인**
- 한지 질감 배경에 먹물+채색 일러스트
- 각 월의 식물/동물은 유지하되, 저승적 변형 적용

### 월별 디자인 컨셉

> **방향:** 전통 화투의 구도와 모티프를 충실히 유지하되,
> 저승 세계관 요소를 20% 정도만 은은하게 반영한다.
> 플레이어가 "아, 화투다" 하고 즉시 알아볼 수 있어야 한다.

| 월 | 전통 모티프 (유지) | 저승 트위스트 (은은하게) |
|----|-------------------|----------------------|
| 1월 송학 | 소나무 + 학 | 학 날개 끝 살짝 투명, 소나무 껍질에 미세 해골 암시 |
| 2월 매조 | 매화 + 꾀꼬리 | 꾀꼬리 눈 미세 붉은빛, 꽃잎 일부 핏방울 느낌 |
| 3월 벚꽃 | 벚꽃 + 장막(커튼) | 장막 뒤 희미한 그림자 실루엣 |
| 4월 흑싸리 | 흑싸리 + 두견새 | 흑싸리 그림자가 손가락처럼 뻗는 형태 |
| 5월 난초 | 창포 + 나무다리 | 다리 아래 물이 삼도천 암시 |
| 6월 모란 | 모란 + 나비 | 나비 날개에 해골 무늬 힌트 |
| 7월 홍싸리 | 홍싸리 + 멧돼지 | 멧돼지 눈 붉은 빛, 도깨비불 점 |
| 8월 공산 | 억새 + 보름달 | 달 표면에 미세한 해골 문양 |
| 9월 국진 | 국화 + 술잔 | 술잔 안 액체 은빛 인광 |
| 10월 단풍 | 단풍 + 사슴 | 사슴 눈 공허한 흰색 |
| 11월 오동 | 오동 + 봉황 | 봉황 깃털 끝 저승 푸른 불꽃 |
| 12월 비 | 비 + 우산 사내(저승사자) | 저승사자 복장(검은 갓+도포) |

### 카드 타입별 프레임
```
┌────────────┐     ┌────────────┐
│ ★ 광 ★    │     │ ═ 띠 ═    │
│            │     │            │
│  [일러스트]  │     │  [일러스트]  │
│            │     │            │
│  1월 송학   │     │  1월 홍단   │
│ ────────── │     │ ────────── │
│  금테두리   │     │  빨간테두리 │
└────────────┘     └────────────┘

┌────────────┐     ┌────────────┐
│ ◆ 열끗 ◆  │     │ ● 피 ●    │
│            │     │            │
│  [일러스트]  │     │  [일러스트]  │
│            │     │            │
│  2월 매조   │     │  1월 피    │
│ ────────── │     │ ────────── │
│  파란테두리 │     │  회색테두리 │
└────────────┘     └────────────┘
```

### 카드 뒷면
- 한지 크림 배경 + 한국 전통 문양(태극, 팔괘)
- 네 귀퉁이에 도깨비불(네온 청색) 장식
- 전통 기하학 테두리 문양(먹색+금색)
- 전체 패 동일 디자인

## 6. 사운드 디자인

### BGM 스타일: 국악 × 일렉트로닉 퓨전

**악기 구성**
| 악기 | 역할 | 비고 |
|------|------|------|
| 거문고 | 메인 멜로디, 긴장감 | 저음의 웅장함 |
| 가야금 | 서브 멜로디, 감성 | 고음의 청아함 |
| 장구 | 리듬 | 화투판의 박자감 |
| 대금 | 배경, 분위기 | 쓸쓸함/신비로움 |
| 신스(Synth) | 저음, 패드 | 현대적 층 |
| 일렉 비트 | 보스전 리듬 | 긴박감 |

### 화면별 BGM

| 화면 | BPM | 분위기 | 주 악기 |
|------|-----|--------|---------|
| 메인 메뉴 | 70 | 쓸쓸하고 신비로움 | 대금 + 신스 패드 |
| 일반 라운드 | 100 | 긴장감 있지만 여유 | 거문고 + 장구 |
| 보스전 | 130 | 격렬하고 긴박 | 일렉 비트 + 거문고 |
| Go 선택 | 140 | 심장 두근거림 | 장구 가속 + 신스 |
| 상점 | 80 | 편안하고 기묘 | 가야금 + 대금 |
| 이벤트 | 90 | 미스터리 | 대금 솔로 + 앰비언트 |
| 승리 | 120 | 환희, 해방감 | 전체 합주 |
| 패배 | 60 | 절망, 슬픔 | 거문고 솔로 + 리버브 |

### 현재 구현 상태 (Love2D 프로토타입)

**BGM**
- CC0 라이선스 5트랙 사용 (OpenGameArt 출처)
- 게임 상태(메뉴/전투/상점/이벤트/승리) 기반 자동 전환
- 1초 크로스페이드로 부드러운 전환
- 설정에서 BGM 볼륨 슬라이더 제어 (1% 단위)

**SFX**
- 절차적 생성 방식: 외부 파일 없이 `sfx.lua`에서 24종 효과음 코드 생성
- 카드/족보/보스/UI 등 전 영역 커버

**설정 아이콘**
- `icon_generator.lua`로 16x16 픽셀아트 도트 아이콘 절차적 생성
- 지구본(언어), 번개(속도), 흔들림 파형(화면흔들림), 음표(BGM), 스피커(SFX)
