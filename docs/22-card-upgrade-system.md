# 화투패 업그레이드 & 이펙트 시스템

## 1. 개요

### 핵심 차별점
> 화투패 자체를 **강화/변이/각성**시키는 시스템.
> Balatro에서는 카드를 바꾸지만, 우리는 카드를 **진화**시킨다.
> 48장의 고정된 덱이 플레이할수록 **나만의 덱**이 된다.

---

## 2. 카드 강화 시스템 (Card Enhancement)

### 강화 등급
```
[기본] → [연마(鍊磨)] → [신통(神通)] → [전설(傳說)] → [해탈(解脫)]
 ★         ★★           ★★★          ★★★★         ★★★★★
```

### 강화 방법
| 방법 | 비용 | 설명 |
|------|------|------|
| 대장간 (저승 장터) | 엽전 50~200 | 카드 1장을 다음 등급으로 강화 |
| 전투 보상 | 무료 | 판에서 최다 사용 카드 자동 강화 확률 |
| 부적 효과 | - | 특정 부적이 조건부 자동 강화 |
| 이벤트 | - | 이벤트 선택지로 강화 |

### 등급별 효과

**연마(★★) — 기본 수치 상승**
| 카드 타입 | 연마 보너스 |
|-----------|-----------|
| 광 | 칩 +10 |
| 띠 | 칩 +5 |
| 열끗 | 칩 +5 |
| 피 | 피 가치 +1 (쌍피 효과) |

**신통(★★★) — 특수 능력 부여**
| 카드 타입 | 신통 능력 |
|-----------|----------|
| 광 | 매칭 시 배수 +1 (데미지 증폭) |
| 띠 | 같은 색 띠 매칭 시 칩 +15 (족보 데미지 강화) |
| 열끗 | 매칭 시 다음 카드 드로우 |
| 피 | 매칭 시 30% 확률로 복제 (피 +1) |

**전설(★★★★) — 고유 이펙트**
| 카드 | 전설 이펙트 |
|------|-----------|
| 1월 광 (송학) | '학의 춤': 매칭 시 모든 광 칩 +20 |
| 3월 광 (벚꽃 커튼) | '꽃비': 매칭 시 바닥패 전체 공개 3초 |
| 8월 광 (달) | '월하': 이 패로 매칭 시 배수 ×1.5 |
| 11월 광 (봉황) | '봉황 날개': 매칭 시 손패 전체 1등급 일시 강화 |
| 12월 광 (비) | '폭우': 비광 패널티 제거, 삼광으로 판정 |

**해탈(★★★★★) — 초월 이펙트 (무한 모드 전용)**
| 효과 | 설명 |
|------|------|
| 초월 광 | 광 족보 칩/배수 2배 (데미지 2배) |
| 초월 띠 | 띠 1장이 모든 색 띠로 판정 |
| 초월 열끗 | 열끗 3장만으로 족보 성립 |
| 초월 피 | 피 5장만으로 족보 성립 |

## 3. 카드 변이 시스템 (Card Mutation)

### 컨셉
> 카드의 **월(月)이나 타입(種類)**을 바꾸는 시스템.
> 기존 화투 족보의 한계를 깨고 창의적 빌드를 가능하게.

### 변이 종류

**월 변이 (Month Shift)**
```
[3월 피] → 월 변이 → [8월 피]
→ 8월에 피가 3장이 됨 → 매칭 확률 증가
→ 특정 월에 카드를 집중시켜 쓸(TripleMatch) 확률 극대화
```

**타입 변이 (Type Shift)**
```
[1월 피] → 타입 변이 → [1월 띠(홍단)]
→ 홍단이 4장이 됨 → 홍단 족보가 쉬워짐
→ but 피 수 감소 → 피 족보 약화
→ 트레이드오프 결정 필요
```

**융합 변이 (Fusion)**
```
[1월 광] + [8월 광] → 융합 → [태양과 달의 광]
→ 새로운 유일무이한 카드 생성
→ 두 월에 모두 매칭 가능 (이중 월)
→ 칩/배수 보너스
```

### 변이 획득 방법
| 방법 | 빈도 |
|------|------|
| 특수 이벤트 '변이의 샘' | 런 당 1~2회 |
| 전설 부적 '광기의 광' 효과 | 부적 보유 시 |
| 무한 모드 강화 선택 | 웨이브 보상 |
| 알케미스트 장터 (특수 저승 장터) | 엽전 300 |

### 변이 제한
- 한 런에서 최대 **5회** 변이 가능 (밸런스)
- 광 → 피 변이 불가 (하향 변이 금지)
- 융합은 같은 타입끼리만 가능
- 변이된 카드는 시각적으로 '먹물 얼룩' 이펙트 표시

## 4. 카드 이펙트 시스템 (Visual Effects)

### 등급별 시각 이펙트

**기본 (★)**
```
일반 카드. 깨끗한 한지 위에 전통 문양.
이펙트 없음.
```

**연마 (★★)**
```
카드 테두리에 은빛 광택.
매칭 시 은빛 파티클 2~3개.
```

**신통 (★★★)**
```
카드에서 미세한 도깨비불(파란 불꽃) 발산.
매칭 시 불꽃 폭발 이펙트.
카드 표면에 미세한 먹물 문양 애니메이션.
```

**전설 (★★★★)**
```
카드 전체가 금빛으로 빛남.
카드 표면에 홀로그램 효과 (각도에 따라 색 변화).
매칭 시 대형 먹물 파도 이펙트 + 화면 진동.
고유 이펙트 이름 텍스트 표시 ("학의 춤!").
```

**해탈 (★★★★★)**
```
카드가 반투명하게 빛나며 부유하는 느낌.
카드 주변에 피안화 꽃잎 파티클.
매칭 시 화면 전체 먹물 파도 + 초월적 사운드.
카드 뒷면에 윤회 문양이 빛남.
```

### 족보 완성 이펙트

| 족보 | 이펙트 |
|------|--------|
| 홍단 | 붉은 먹물이 화면을 가로지름 + "홍단!" 큰 텍스트 |
| 청단 | 파란 번개 + 먹물 서예 "청단" |
| 초단 | 초록 바람 + 잎사귀 파티클 |
| 삼광 | 세 줄기 금빛 기둥 + 하늘에서 학 비행 |
| 사광 | 사방에서 금빛 폭발 + 화면 백색 플래시 |
| 오광 | 화면 전체가 금색 → 먹물로 "五光" 서예 → 폭죽 파티클 |
| 피 10+ | 붉은 빗방울이 화면에 쏟아짐 |
| 쓸 | 바람이 바닥패를 쓸어감 + "쓸!" 다이나믹 텍스트 |

### Go/Stop 이펙트

**Go 선택**
```
1 Go: 화면 가장자리 붉은 빛 + 심장박동음
2 Go: 화면 붉어짐 강화 + 도깨비불 증가 + 불안한 진동
3 Go: 화면 핏빛 + 균열 이펙트 + 도깨비 웃음소리 반복
      + "욕심이 널 죽인다" 텍스트 깜빡임
```

**Go 성공**
```
점수 카운트업: 숫자가 가속하며 커짐
배수 적용: "×2" "×4" "×10" 순차 표시 + 화면 흔들림 점점 강해짐
최종: 금색 폭발 + 수치 확정
```

**Go 실패 (즉사)**
```
화면 균열 확산 → 핏빛 화면 → 패가 하나씩 사라짐
"욕심이 널 삼켰다..."
도깨비 웃음소리 에코
검은 화면으로 페이드
```

## 5. 덱 커스터마이징

### 덱 편집 (런 사이)
스토리 진행 시 해금되는 **덱 편집 기능**.

```
[덱 편집 화면]

48장 전체 카드 표시
├── 강화 등급 표시 (★~★★★★★)
├── 변이 상태 표시 (원래 월 → 현재 월)
├── 소지 이펙트 목록
│
├── [카드 제거]: 패 1장 제거 (최소 40장)
├── [카드 강화]: 엽전/자원으로 등급 상승
├── [카드 변이]: 월/타입 변경
└── [카드 복원]: 변이/강화 초기화 (무료)
```

### 덱 빌드 아키타입

| 빌드명 | 전략 | 핵심 강화 |
|--------|------|----------|
| 광 집중 | 광 5장 모두 전설 등급 | 광 카드 전부 전설화 + 오광 특화 부적 |
| 피의 바다 | 피 패 극대화 | 피 연마(쌍피화) + 피의 맹세 + 피 강화 부적 |
| 홍단 마스터 | 홍단 확정 + 추가 띠 | 1,2,3월 띠를 신통으로 + 월 변이로 홍단 추가 |
| 쓸 빌드 | 같은 월 집중 | 월 변이로 특정 월에 6~8장 집중 → 매 턴 쓸 |
| Go 올인 | Go 3회 특화 | Go 배수 강화 부적 + 생존 강화 + Go 보험 |
| 균형 | 다양한 족보 | 각 타입 연마 수준 + 범용 부적 |

## 6. 카드 연출 시스템 (기술 사양)

### 이펙트 컴포넌트 구조
```
CardEffectSystem
├── CardGlowEffect        — 카드 테두리 빛남 (등급별)
├── CardParticleEffect     — 카드 주변 파티클 (도깨비불 등)
├── CardHologramShader     — 홀로그램 셰이더 (전설+)
├── MatchExplosionEffect   — 매칭 시 폭발 이펙트
├── YokboCompleteEffect    — 족보 완성 연출
├── GoStopDramaticEffect   — Go/Stop 긴장감 연출
└── MutationVisualEffect   — 변이 카드 먹물 얼룩
```

### 셰이더 목록
| 셰이더 | 용도 | 기술 |
|--------|------|------|
| InkSpread | 먹물 퍼짐 | UV 기반 마스크 애니메이션 |
| HanjiTexture | 한지 질감 | 노이즈 텍스처 + 노멀맵 |
| GhostFire | 도깨비불 | 파티클 + 디스토션 |
| HolographicCard | 카드 홀로그램 | 뷰 앵글 기반 색상 변화 |
| BloodRain | 핏빛 비 | 스크린 스페이스 파티클 |
| GoldenBurst | 금빛 폭발 | 블룸 + 파티클 |
| InkCalligraphy | 서예 텍스트 | 스트로크 애니메이션 |

## 7. 카드 수집 & 도감

### 카드 도감
```
[도감 → 화투패]

12월 전체 카드 표시
├── 보유 여부 (흑백 vs 컬러)
├── 최고 강화 등급 표시
├── 사용 통계 (매칭 횟수, 족보 기여 횟수)
├── 변이 이력
│
├── 카드별 상세:
│   ├── 카드 이름 (한/영)
│   ├── 전통 화투 유래 설명
│   ├── 저승 리디자인 배경 스토리
│   ├── 각 강화 등급의 일러스트 변화 (5단계)
│   └── 이펙트 미리보기
│
└── 수집률: 48/48 (100%)
    강화 완료: 23/48
    전설 달성: 5/48
```

### 카드 스킨 시스템
| 스킨 | 해금 조건 | 설명 |
|------|---------|------|
| 전통 화투 | 기본 | 클래식 화투 디자인 |
| 저승 화투 | 기본 | 저승 테마 리디자인 (기본 아트) |
| 금빛 화투 | 전 족보 완성 업적 | 모든 카드에 금박 테두리 |
| 먹물 화투 | 웨이브 50 달성 | 수묵화 스타일 |
| 핏빛 화투 | 나락 난이도 클리어 | 모든 카드에 붉은 색조 |
| 사이버 화투 | 시크릿 | 네온 + 글리치 스타일 |
