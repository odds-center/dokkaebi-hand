# 부적(Talisman) 시스템 문서

## 개요
부적은 Balatro의 조커(Joker) 역할. 패시브/트리거 효과로 족보 데미지를 증폭시키는 핵심 시너지 시스템.

## 기본 규칙
- **최대 5개 슬롯** 장착 가능
- 저주 부적은 슬롯 제한 무시, 강제 장착, 제거 불가
- 저승 장터에서 엽전으로 구매
- 보스 격파 시 드랍 가능

## 효과 적용 순서
```
1단계: 가산 효과 (+Chips, +Mult)
2단계: 승산 효과 (×Mult)
3단계: 특수 효과 (카드 변이, 소멸, 와일드카드 등)
```

## 트리거 타입
| 트리거 | 설명 |
|--------|------|
| OnCardPlayed | 패를 낼 때 |
| OnYokboComplete | 족보 완성 시 |
| OnTurnStart / End | 턴 시작/종료 |
| OnRoundStart / End | 라운드 시작/종료 |
| OnGoDecision | Go 선택 시 |
| OnStopDecision | Stop 선택 시 |
| OnMatchSuccess / Fail | 매칭 성공/실패 시 |
| Passive | 상시 적용 |

## MVP 부적 (3종)

### 1. 피의 맹세 (일반)
- **효과:** 획득한 피 패 1장당 +1 배수
- **트리거:** OnRoundEnd
- **전략:** 피를 많이 모으는 빌드에 강력. 피 10장 이상 시 칩+배수 동시 확보.

### 2. 도깨비 감투 (희귀)
- **효과:** 보유한 띠 1장당 보스 HP -5%
- **트리거:** Passive
- **전략:** 띠 빌드 시 보스 HP를 크게 낮춰 클리어 용이. 홍단/청단/초단 완성 시 시너지.

### 3. 저승사자의 명부 (전설)
- **효과:** 최종 데미지 끝자리가 4일 때 배수 ×4
- **트리거:** OnRoundEnd
- **전략:** 데미지 끝자리를 4로 맞추는 고급 전략 필요. 다른 부적과 조합하여 의도적으로 조정 가능.

## 확장 예정 부적 (Alpha 단계)

| 등급 | 이름 | 효과 |
|------|------|------|
| 일반 | 홍살문 | 홍단 완성 시 +30 칩 |
| 희귀 | 달빛 여우 | 매칭 실패 시 50% 와일드카드 변환 |
| 전설 | 광기의 광 | 광 사용 시 랜덤 패 1장 광으로 변이 |
| 저주 | 흉살 | 매 턴 피 1장 자동 소멸 (강제 장착) |

## 시너지 예시
- **피의 맹세 + 피 10장 족보:** 칩 30 + 배수 +10 → 큰 데미지 확보
- **도깨비 감투 + 홍단/청단 콤보:** 보스 HP -15%~25% 감소 + 띠 족보 칩
- **저승사자의 명부 + Go 시스템:** 배수 폭발과 끝자리 4 조합 시 x40 이상 가능
