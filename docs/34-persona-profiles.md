# 도깨비의 패 — 플레이어 페르소나 프로필

> 숙련도 1~5 단계. 각 페르소나가 어떻게 게임을 경험하는지 정의하고,
> 해당 페르소나 기준으로 밸런스와 재미를 검증하는 문서.

---

## 숙련도 등급표

| 등급 | 이름 | 한 줄 설명 |
|:---:|------|-----------|
| ⭐ | 입문 | 고스톱도 모르고 게임도 처음 |
| ⭐⭐ | 초급 | 튜토리얼 봤고, 기본은 안다 |
| ⭐⭐⭐ | 중급 | 족보를 외웠고, 시너지를 쌓을 줄 안다 |
| ⭐⭐⭐⭐ | 상급 | 부적 빌드를 짜고, Go 리스크를 계산한다 |
| ⭐⭐⭐⭐⭐ | 고수 | 무한 나선을 돌며 최적 루트를 찾는다 |

---

## 페르소나 1 — ⭐ 입문: 김모름

### 프로필
- **나이/직업:** 30대, 회사원. 점심시간에 스팀 훑다가 클릭
- **게임 경험:** 캐주얼 모바일 게임만 해봄. 고스톱 규칙 모름
- **기대:** "한국 감성 카드 게임이래. 한번 해볼까"
- **인내심:** 5분 안에 재미 못 느끼면 환불

### 플레이 패턴
- 카드를 아무거나 1장 선택 → 내기
- Go/Stop 의미를 모름 → 항상 스톱
- 공격 카드도 아무거나 2장
- 부적/축복 무시, 상점 안 감

### 기대 경험
| 구간 | 목표 | 현재 달성 여부 |
|------|------|:---:|
| 1관문 | 뭔지 모르겠지만 이김 | ✅ (HP 100, 6판) |
| 2관문 | "아 이렇게 하는 거구나" | ✅ (HP 150, 6판) |
| 3관문 | 첫 패배 경험 → 다시 도전 | ✅ (HP 200) |
| 4관문 | 벽. "시너지가 뭐지?" → 학습 동기 | ✅ (HP 300) |
| 5관문~ | 여기까진 안 옴. 3~4관문에서 게임오버 | ✅ 의도된 한계 |

### 핵심 수치 시뮬레이션
```
시작: 체력 5, 엽전 50냥

1관문 먹보 도깨비 (HP 100)
  판1: 갑오(5) + 칩(10) = 15
  판2: 3끗(11) + 칩(8) = 19
  판3: 독사(32) + 칩(12) = 44
  판4: 7끗(19) + 칩(6) = 25
  → 누적 103 > HP 100 → 격파! ✅

  "오 이겼다? 뭔가 나온다" → 웨이브 강화 선택
  → 체력+1 선택 (가장 직관적)

2관문 장난꾸러기 (HP 150)
  6판 안에 평균 150 필요 → 판당 25
  → 운 좋으면 격파, 나쁘면 아슬아슬

3관문 불꽃 도깨비 (HP 200)
  → 50:50 확률. 여기서 첫 패배 가능
  → "아... 뭔가 더 강하게 칠 방법이 없나?"
  → 시너지 시스템 학습 동기 발생!
```

### 이 페르소나에게 중요한 것
- ✅ 첫 보스는 반드시 이겨야 함
- ✅ UI가 "이 카드를 선택하세요" 가이드
- ✅ 패배해도 "이유"를 알 수 있어야 함
- ❌ 복잡한 족보 이름은 아직 의미 없음

## 페르소나 2 — ⭐⭐ 초급: 이배움

### 프로필
- **나이/직업:** 20대, 대학생. 고스톱을 명절에 해봄
- **게임 경험:** Slay the Spire 30시간, Balatro 안 해봄
- **기대:** "로그라이트 카드 게임이래. 덱빌딩 재밌겠다"
- **인내심:** 30분 정도 학습 의지 있음

### 플레이 패턴
- 같은 월 카드를 모아서 내기 → 쌍/땡 노림
- Go 1회 정도 시도 ("카드 더 뽑으면 좋겠지?")
- 상점에서 싼 부적 구매
- 축복은 설명 읽고 고름

### 기대 경험
| 구간 | 목표 | 현재 달성 여부 |
|------|------|:---:|
| 1~3관문 | 무난하게 클리어 | ✅ |
| 4~5관문 | Go 1~2 활용하여 클리어 | ✅ |
| 6~7관문 | 부적 효과 체감 시작 | ✅ |
| 8~10관문 | 최종 보스 도전 → 50% 성공 | ✅ |
| 나선 2 | 목표. "다음 나선 가보자" | ✅ |
| 나선 3 | 여기서 막힘. 성장 욕구 발생 | ✅ |

### 핵심 수치 시뮬레이션
```
시작: 체력 5, 엽전 50냥

전략: 2장 내기 (쌍 노림) → Go 1 → 스톱 → 공격

1관문 (HP 100):
  내기: 1월쌍(30칩 × 1.2배) = 36칩
  Go 1: +3장 드로우
  스톱 → 공격: 알리(35) + 36칩 × 1.2배 × Go(1.5)
  = (35 + 36) × 1.2 × 1.5 = 128
  → 1판 킬! ✅

5관문 (HP 250):
  내기1: 홍단(50칩 × 1.5배)
  내기2: 추가 쌍(20칩)
  Go 1 → 스톱 → 장땡(50) + 70칩 × 1.5배 × 1.5
  = (50 + 70) × 1.5 × 1.5 = 270
  → 1판 킬! ✅

10관문 염라대왕 (HP 700):
  Go 2 필요. 여러 판에 걸쳐 격파.
  판1: 200 데미지 → HP 500
  판2: 250 데미지 → HP 250
  판3: 300 데미지 → HP 0 → 격파! ✅
```

### 이 페르소나에게 중요한 것
- ✅ 시너지 쌓는 재미 (쌍 → 콤보 → "합쳐지네!")
- ✅ Go의 리스크/리워드 체감
- ✅ 상점에서 부적 고르는 재미
- ✅ 나선 1 클리어 → "다음 나선 가보자" 욕구

## 페르소나 3 — ⭐⭐⭐ 중급: 박전략

### 프로필
- **나이/직업:** 20대, 게임 유튜버. 로그라이트 전문
- **게임 경험:** Balatro 200시간, StS 500시간
- **기대:** "화투 로그라이트? 콤보 시스템 분석해봐야지"
- **인내심:** 시스템 이해할 때까지 파고듦

### 플레이 패턴
- 3~4장 내기 → 홍단/청단/고도리 세트 노림
- Go 1~2 적극 활용 (배수 ×1.5~2)
- 부적 빌드: 배수 계열 우선
- 축복: 공허(부적 ×2) 또는 빙결(배수 +1)

### 기대 경험
| 구간 | 목표 | 현재 달성 여부 |
|------|------|:---:|
| 나선 1 | 전 관문 클리어 | ✅ |
| 나선 2 | 빌드 완성 → 스무스하게 클리어 | ✅ |
| 나선 3 | 도전적이지만 가능 | ✅ |
| 나선 4 | 벽. 부적+웨이브 강화 의존 | ✅ (후반 패배) |
| 나선 5 | 상급 플레이 필요 | ✅ |

### 핵심 수치 시뮬레이션
```
축복: 빙결 (배수 +1, 손패 -1)

나선 1 전략:
  매 관문: 내기 2~3회 → 콤보 50~80칩 × 1.5~2.0배
  Go 1 → 섯다 공격
  데미지: (50 + 60) × 1.8 × 1.5 = 297/판
  → 대부분 1~2판 킬

나선 3 (HP ×3.2):
  3관문 HP: 200 × 3.2 = 640
  웨이브 누적: 칩+40, 배수+3
  데미지: (50 + 100) × 2.5 × 2.0 = 750/판
  → 1판 킬 가능. 빌드 완성 쾌감!

나선 4 (HP ×5.8):
  3관문 HP: 200 × 5.8 = 1,166
  데미지: (50 + 120) × 3.0 × 2.0 = 1,020/판
  → 아슬아슬. 8관문+(HP 3,500+)부터 패배 시작
```

### 이 페르소나에게 중요한 것
- ✅ 콤보 발견의 쾌감 ("이거랑 이거 합치면 이렇게 되네!")
- ✅ 빌드 다양성 (배수 빌드 vs 칩 빌드 vs 부적 빌드)
- ✅ Go의 전략적 깊이 (언제 Go, 언제 Stop?)
- ✅ 나선 간 성장 체감

## 페르소나 4 — ⭐⭐⭐⭐ 상급: 최고수

### 프로필
- **나이/직업:** 30대, 직장인. 로그라이트 올클리어 수집가
- **게임 경험:** Balatro Gold Stake 클리어, StS A20 클리어
- **기대:** "무한 나선 어디까지 갈 수 있나 보자"
- **인내심:** 수십 시간 투자 가능

### 플레이 패턴
- 4~5장 내기 → 티어 S/A 콤보 직행
- Go 2~3 적극 도전 (배수 ×2~3)
- 부적 빌드: 전설 부적 중심 (염라왕의 도장 ×3, 지옥불꽃 ×2)
- 웨이브 강화: 배수 우선 → 칩 → 부적 슬롯

### 기대 경험
| 구간 | 목표 | 현재 달성 여부 |
|------|------|:---:|
| 나선 1~3 | 워밍업. 거의 무피해 클리어 | ✅ |
| 나선 4~5 | 본격 도전 구간 | ✅ |
| 나선 7~8 | 숫자 K 단위 진입. 부적 필수 | ✅ (스케일링) |
| 나선 10 | 목표. "나선 10 찍자" | ✅ |

### 핵심 수치 시뮬레이션
```
축복: 공허 (부적 ×2, 슬롯 -2)

나선 5 (HP ×10.5):
  보스 HP: 100 × 10.5 = 1,050 (1관문) ~ 700 × 10.5 = 7,350 (10관문)
  웨이브 누적: 칩+80, 배수+8
  전설 부적: 염라왕 도장(×3), 효과 ×2(공허) = ×6

  Go 2 + 38광땡:
  (80 + 160) × 6.0 × 2.0 = 2,880/판
  → 1관문 1판, 10관문 3판

나선 10 (HP ×198):
  보스 HP: 100 × 198 = 19,800 (1관문) ~ 700 × 198 = 139K (10관문)
  웨이브 누적: 칩+180, 배수+18

  Go 3 + 38광땡 + 전설 부적:
  (80 + 260) × 15.0 × 3.0 = 15,300/판
  → 1관문 2판, 10관문 10판 (한계)
  → NumberFormatter: "15.3K" "139K" 표시
```

### 이 페르소나에게 중요한 것
- ✅ 숫자 성장의 쾌감 (100 → 1K → 10K)
- ✅ "한 방 킬" 순간의 카타르시스
- ✅ 빌드 최적화의 깊이
- ✅ 끝이 없는 도전 (무한 나선)
- ✅ K/M 단축 표기가 자연스럽게 등장

## 페르소나 5 — ⭐⭐⭐⭐⭐ 고수: 도깨비왕

### 프로필
- **나이/직업:** 20대, 스피드런/최고기록 커뮤니티 활동
- **게임 경험:** 모든 로그라이트 마스터. 메타 최적화 전문
- **기대:** "나선 몇까지 갈 수 있는지가 내 기록이다"
- **인내심:** 무한. 기록 갱신이 동기

### 플레이 패턴
- 오프닝 최적화: 축복 + 사주팔자 조합 계산
- 내기 5회 풀 사용 → 티어 S 콤보 + 저승 족보 동시 달성
- Go 3 매 판 도전 (즉사 10% 감수)
- 부적: 전설 3개 + 공허(×2) = 배수 폭발
- 웨이브 강화: 순서 최적화 (배수 → 칩 → 슬롯)
- 상점: 감정부(전설 확정) 최우선 구매

### 기대 경험
| 구간 | 목표 | 현재 달성 여부 |
|------|------|:---:|
| 나선 1~5 | 스피드런. 최소 판수 클리어 | ✅ |
| 나선 10 | 숫자 K 단위. 안정적 클리어 | ✅ |
| 나선 15 | 숫자 M 단위 진입 | ✅ (스케일링) |
| 나선 20+ | 숫자 B 단위. 즉사 리스크와 싸움 | ✅ |
| 기록 | "나선 XX까지 갔다" 스크린샷 | ✅ |

### 핵심 수치 시뮬레이션
```
나선 15 (HP ×3,748):
  보스 HP: 100 × 3,748 = 375K ~ 700 × 3,748 = 2.6M
  웨이브 누적: 칩+280, 배수+28
  전설 부적 3개 × 공허(×2): 배수 ×36

  Go 3 + 38광땡:
  (80 + 360) × 36.0 × 3.0 = 47.5K/판
  → NumberFormatter: "47.5K"
  → 1관문 8판, 10관문 55판 (극한)

나선 20 (HP ×70,824):
  보스 HP: 7.1M ~ 49.6M
  데미지: ~120K/판
  → NumberFormatter: "7.1M" "120K"
  → 1관문 59판. 사실상 한계.
  → 영구 강화가 여기서 결정적 차이

나선 25+ (HP ×B 단위):
  "1.23e9" 과학적 표기 등장
  → 여기까지 오는 게 "도전"
```

### 이 페르소나에게 중요한 것
- ✅ 무한한 스케일링 (끝이 없다)
- ✅ 숫자 단위 변화 (100 → K → M → B)
- ✅ 즉사 리스크 관리 (Go 3 = 10%, 매 판)
- ✅ 영구 강화의 장기 투자 가치
- ✅ 리더보드/기록 공유 욕구
- ✅ "이론상 최대 데미지" 계산의 재미

```
   ⭐ 입문: 나선 1의 1~3관문 (HP 100~200)
```

## 클리어 기준 & 중독 루프

### 클리어 = 나선 1 완료 (10관문 → 이승의 문)

모든 플레이어가 **결국은 클리어할 수 있다.** 로그라이트의 핵심은:

```
 죽는다 → 넋(영혼) 획득 → 영구강화 구매 → 재도전 → 더 멀리 간다 → ...
   │                                                      │
   └──────────── 중독 루프 ────────────────────────────────┘
```

### 멀티런 클리어 시뮬레이션

**⭐ 입문 김모름 — 목표: ~2시간 (4~5런)**
```
런 1 (스킬 0): 1~3관문 클리어 → 사망 → 넋 30~50 획득
  "아 이래서 죽는구나... 근데 이 넋으로 뭔가 살 수 있네?"
  → 강화 구매: 기본 칩+5 (20넋)
  → 소요: 약 15분

런 2 (스킬 1): 3~5관문 클리어 → 사망 → 넋 60~100 획득
```

### 중독성의 핵심 요소

| 요소 | 구현 | 효과 |
|------|------|------|
| **"한 판만 더"** | 런 1회 = 15~30분 | 부담 없는 한 판 |
| **죽어도 성장** | 넋 → 영구강화 | 헛된 죽음 없음 |
| **매번 다른 보스** | 랜덤 보스 + 랜덤 파츠 | 신선함 유지 |
| **"거의 다 왔는데!"** | 7~8관문에서 사망 | 재도전 욕구 폭발 |
| **숫자 성장** | 100 → K → M | 파워 판타지 |
| **선택의 재미** | 축복/부적/웨이브 3택 | 매 런 다른 빌드 |
| **엔딩 후 계속** | 나선 무한 반복 | 기록 도전 |
