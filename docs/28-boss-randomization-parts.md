# 보스 랜덤화 & 파츠 시스템

## 1. 개요

### 핵심 컨셉
> 보스는 더 이상 고정된 존재가 아니다.
> **랜덤 조합 + 파츠 장착**으로 매 런마다 다른 보스를 만난다.
> 같은 "먹보 도깨비"도 **뿔이 달리면** 다른 놈이다.

### 시스템 구조
```
보스 = [기본 도깨비] + [파츠 0~3개] + [랜덤 기믹 변형]

예시:
  먹보 도깨비 + [철갑 파츠] + [화염 뿔 파츠]
  = "철갑 화염 먹보 도깨비"
  = 기본 기믹(패 소멸) + 방어(첫 3턴 데미지 -50%) + 화염(매 4턴 바닥 리셋)
```

## 2. 기본 도깨비 풀 (Base Pool)

### 스토리 도깨비 (10종 — 항상 랜덤 등장)
십왕 관문의 보스. **모든 모드에서 랜덤 등장 + 랜덤 기물(파츠) 장착.**
보스는 항상 랜덤이며, HP 기반 전투로 족보 데미지를 쌓아 HP를 0으로 만들면 격파.

## 3. 파츠 시스템 (Parts System)

### 파츠란?
도깨비의 **신체 부위에 장착되는 강화 요소**.
시각적으로 도깨비 외형이 변하고, 추가 기믹이 부여된다.

### 파츠 슬롯
```
도깨비 신체:
         [머리 파츠] ← 뿔, 왕관, 투구 등
             │
    [왼팔 파츠]─[몸통 파츠]─[오른팔 파츠]
             │
         [하체 파츠] ← 꼬리, 발굽 등 (Alpha 확장)

스토리 모드:
  관문 1~3: 기물 0개 (기본 도깨비)
  관문 4~6: 기물 1개 (약간 강화)
  관문 7~9: 기물 2개 (상당히 강화)
  관문 10:  기물 3개 (최대 강화)

무한 모드:
  웨이브 1~10:  파츠 0~1개
  웨이브 11~25: 파츠 1~2개
  웨이브 26~50: 파츠 2~3개
  웨이브 51+:   파츠 3개 + 파츠 등급 상승

윤회 모드:
  윤회 횟수만큼 최소 파츠 수 증가
```

## 4. 파츠 조합 시너지

### 세트 효과
같은 테마의 파츠를 2개 이상 장착하면 세트 효과 발동.

| 세트 | 파츠 조합 | 세트 효과 |
|------|----------|----------|
| 화염 세트 | 화염 뿔 + 불꽃 장갑 + 화문 | 매 2턴 바닥패 전체 소각 + 보스 HP +50% |
| 얼음 세트 | 얼음 왕관 + 빙결 갑옷 | 매 턴 손패 2장 동결 + Go 배수 -2 |
| 그림자 세트 | 그림자 팔 + 그림자 갑옷 | 보스 모습 완전 은폐 (체력바만 표시) |
| 해골 세트 | 해골 면류관 + 뼈 집게 | 매 2턴 획득패 2장 해골화 + 쓸 불가 |
| 봉인 세트 | 도깨비불 투구 + 봉인 부적 팔 | 부적 발동률 -50% + 저주 부적 강제 1개 |

### 세트 시각 변화
세트 효과 발동 시 보스의 **오라(Aura)**가 변한다.

```
화염 세트: 보스 주변에 화염 오라 + 바닥에 용암 균열
얼음 세트: 보스 주변에 서리 파티클 + 테이블에 얼음
그림자 세트: 보스가 그림자 속에 완전 은폐 + 눈만 빛남
해골 세트: 보스 주변에 해골 파티클 + 뼈 소리
봉인 세트: 보스 주변에 부적 회오리 + 부적 깨지는 소리
```

## 5. 보스 랜덤 등장 시스템

### 스토리 모드 — 항상 랜덤
```
관문 1~10: 보스 항상 랜덤 생성 (랜덤 도깨비 + 랜덤 기물 조합)
모든 보스는 HP 기반. 족보 데미지로 HP를 0으로 만들면 격파.
보스 반격 시스템: 판이 끝날 때마다 보스가 카운터어택.

예시:
  관문 7 → 랜덤: 태산왕 + [화염 뿔] + [철갑]
  = "화염 철갑 태산왕" (HP: 450)
  = 기본 기믹(저승 장터 가격 2배) + 바닥패 소각 + 초반 방어
  → 매 런마다 다른 보스, 다른 난이도/전략
```

### 무한 모드 — 완전 랜덤
```
def generate_boss(wave):
    # 기본 도깨비 랜덤 선택
    base = random.choice(ALL_DOKKAEBI)

    # 웨이브에 따른 파츠 수
    parts_count = min(3, wave // 10)

    # 파츠 등급 조정
    if wave < 25:
        parts_pool = COMMON + RARE
    elif wave < 50:
        parts_pool = RARE + LEGENDARY
    else:
        parts_pool = LEGENDARY

    # 파츠 랜덤 선택 (슬롯별 1개)
    parts = random_unique(parts_pool, parts_count)

    # 보스 HP 스케일링
    hp = base_hp × (1.15 ^ wave) × (1 + 0.1 × parts_count)

    # 이름 생성
    name = parts_prefix + base.name

    return Boss(base, parts, target, name)
```

### 이름 생성 규칙
```
파츠 0개: "먹보 도깨비"
파츠 1개: "화염 먹보 도깨비"
파츠 2개: "화염 철갑 먹보 도깨비"
파츠 3개: "화염 철갑 해골 먹보 도깨비"

세트 발동: "불의 군주 먹보 도깨비" (세트명으로 대체)
```

## 6. 보스 비주얼 — 파츠 레이어링

### 스프라이트 구조
```
도깨비 1체 = 레이어 합성

Layer 0: 기본 몸체 (체형별)
Layer 1: 머리 파츠 (교체 가능)
Layer 2: 왼팔 파츠 (교체 가능)
Layer 3: 오른팔 파츠 (교체 가능)
Layer 4: 몸통 파츠 (교체 가능)
Layer 5: 오라/이펙트 (세트 효과)
Layer 6: 표정 (감정 상태)

총 스프라이트:
  기본 몸체: 10종 (보스) + 8종 (잡졸) + 5종 (엘리트) = 23종
  머리 파츠: 8종
  팔 파츠: 8종
  몸통 파츠: 8종
  오라: 5종 (세트)
  표정: 4종 (기본/놀람/분노/공포)

조합 가능 수: 23 × 8 × 8 × 8 = 11,776가지 외형
```

### 파츠별 시각 예시

**먹보 도깨비 + 화염 뿔**
```
기본 먹보: 뚱뚱한 붉은 몸, 큰 입, 배에 화투 문양
+ 화염 뿔: 머리 양옆에 불타는 뿔 2개 추가
           뿔에서 불꽃 파티클 지속 발생
           눈동자가 주황→붉은색으로 변경
```

**여우 도깨비 + 해골 면류관 + 그림자 팔**
```
기본 여우: 날씬하고 교활한 모습, 여우 귀, 꼬리
+ 해골 면류관: 여우 귀 사이에 작은 해골 왕관
               왕관에서 보라색 안개 발산
+ 그림자 팔: 한쪽 팔이 그림자 촉수로 변형
             촉수가 미세하게 움직이는 애니메이션
→ "해골 그림자 여우 도깨비" — 환영 + 해골화 + 카드 약탈
```

## 7. 레어 보스 & 시크릿 보스

### 레어 보스 (1% 확률 등장)
특정 조건에서만 극히 드물게 등장하는 특수 보스.

| 레어 보스 | 등장 조건 | 고유 기믹 | 보상 |
|----------|----------|----------|------|
| 금빛 도깨비 | 엽전 300+ 보유 시 5% | 모든 패가 금빛 (족보 데미지 2배 but HP도 2배) | 엽전 500 + 금빛 스킨 |
| 아기 도깨비 | 웨이브 1에서 1% | 기믹 없음, HP 50 | 부적 슬롯 +1 (영구) |
| 천년 도깨비 | 웨이브 100+ | 모든 기믹 동시 + 파츠 5개 | 해탈 등급 카드 + 칭호 |
| 거꾸로 도깨비 | 사주 대흉 시 10% | 모든 룰 반전 (피=광, 광=피) | 특수 각인 "역전" |

### 시크릿 보스: 뱃사공
스토리 트루 엔딩 후, 특정 조건을 만족하면 등장.

```
조건: 윤회 3회 이상 + 모든 도깨비 동료 확보 + 엽전 0

뱃사공: "...이제 알았나. 나도 이 게임의 일부라는 걸."
뱃사공: "마지막 한 판이다. 나를 이겨라."

기믹: '항해자'
- 뱃사공은 플레이어의 모든 강화/부적/각인을 복사
- 플레이어와 완전히 동일한 조건에서 대결
- 순수 실력 대결

HP: 플레이어와 동일 (서로 데미지 경쟁)
보상: "뱃사공의 삿갓" 전용 부적 (유일무이)
      → 효과: 매 런 시작 시 미래의 이벤트 3개를 미리 볼 수 있음
```

| 0 | 기본 | 기본 |
| 1 | +20% | +25% |
| 2 | +50% | +60% |
| 3 | +100% | +120% |
| 3 + 세트 | +150% | +200% |
