## 2. 고유 시스템 상세

### 2.1 운명의 화투 (Fate Hwatu) — 카드 진화 시스템

**Balatro와의 차이:**
- Balatro: 카드를 교체하거나 버림 (카드 자체는 변하지 않음)
- **도깨비의 패: 카드 자체가 강화/변이/각성하여 나만의 카드가 됨**

**핵심 경험:**
```
1월 광 "송학"
  → 연마: 칩 +10
  → 신통: 매칭 시 배수 +1
  → 전설: "학의 춤" — 매칭 시 모든 광 칩 +20
  → 해탈: 초월 광 — 광 족보 칩/배수 2배

같은 "1월 광"이지만, 플레이어마다 전혀 다른 카드가 됨.
100시간 플레이한 사람의 1월 광 ≠ 1시간 플레이한 사람의 1월 광
→ "내 카드"라는 애착 + 장기 성장 동기
```

**왜 차별적인가:**
- 카드 게임에서 "카드 자체의 성장"은 거의 없는 요소
- TCG의 "포일/레어 카드" 수집 욕구 + RPG의 "장비 강화" 쾌감 결합
- 48장이라는 고정 덱이 "나만의 48장"이 되는 경험

### 2.2 도깨비 각인 (Dokkaebi Seal) — 카드에 영혼을 새기는 시스템

**컨셉:**
> 격파한 도깨비의 **영혼을 카드에 새겨** 특수 효과를 부여.
> 보스마다 고유 각인이 있으며, 같은 카드에 다른 각인을 조합 가능.

**각인 종류 (10종 — 십왕 대응)**

| 각인 | 출처 | 카드 적용 효과 |
|------|------|-------------|
| 탐식의 각인 | 먹보 도깨비 | 매칭 시 보스 HP -10 |
| 기만의 각인 | 장난꾸러기 | 이 카드가 2개의 월에 동시 소속 |
| 환혹의 각인 | 여우 도깨비 | 매칭 실패 시에도 칩 +5 |
| 진실의 각인 | 거울 도깨비 | 반전 판에서도 정상 데미지 |
| 심판의 각인 | 염라왕 | 이 카드로 족보 완성 시 배수 +3 |
| 분노의 각인 | 변성왕 | 연속 매칭 시 칩 +10 누적 |
| 탐욕의 각인 | 태산왕 | 매칭 시 엽전 +5 |
| 인내의 각인 | 평등왕 | 마지막 턴에 사용 시 칩/배수 2배 |
| 복제의 각인 | 도시왕 | 매칭 시 30% 확률로 같은 효과 복제 |
| 윤회의 각인 | 전륜왕 | 런 종료 시 이 카드 강화 등급 유지 |

**적용 규칙:**
- 카드 1장에 최대 **2개** 각인 가능
- 같은 각인 중첩 불가
- 각인 제거: 상점에서 엽전 100
- 각인 조합 시너지 존재 (아래)

**각인 시너지 (조합 보너스)**
| 조합 | 효과 |
|------|------|
| 탐식 + 탐욕 | 매칭 시 보스 HP -10 & 엽전 +5 **+ 추가 칩 +20** |
| 기만 + 환혹 | 이중 월 매칭 실패 시에도 칩 +10 **+ 와일드카드화** |
| 심판 + 진실 | 족보 완성 시 배수 +3 **+ 반전 면역 + 데미지 추가 +2배** |
| 분노 + 인내 | 마지막 턴 연속 매칭 시 **칩/배수 3배** |
| 복제 + 윤회 | **복제 확률 60%로 증가 + 각인도 다음 런 유지** |

### 2.3 욕망의 저울 (Greed Scale) — 리스크 시각화

**컨셉:**
> 화면에 항상 보이는 **욕망의 저울** UI.
> Go를 선택할수록 저울이 기울어지며, 시각적으로 리스크를 체감.

```
Stop 쪽                              Go 쪽
   ◇                                  ◇
   │                                  │
   │         ┌────┐                   │
   │    ─────│균형│─────              │
   │         └────┘                   │
  [안전]      ↕↕↕              [욕망]
  배수 x1    저울 중립          배수 x10

Go 1회:
   ◇                                  ◇
   │              ┌────┐              │
   │         ─────│    │──────        │
   │              └────┘              │
  [안전]       기울어짐 →      [욕망↑]

Go 3회:
   ◇                                  ◇
   │                    ┌────┐        │
   │         ───────────│!!! │        │
   │                    └────┘        │
  [안전]           위험!! →    [극한 욕망]
  저울이 심하게 기울어지며 붉게 빛남
  도깨비불이 미친듯이 타오름
```

**왜 차별적인가:**
- Go/Stop의 리스크가 단순 텍스트가 아닌 **감정적 경험**이 됨
- 저울이 기울수록 BGM 변화, 화면 색조 변화, 파티클 증가
- "한 번만 더" 심리를 시각적으로 극대화
- 다른 어떤 카드 게임에도 없는 고유 UI 요소

### 2.4 혼령 소환 (Spirit Summon) — 길들인 도깨비

**컨셉:**
> 격파한 도깨비를 **길들여 소환**하여 전투 중 능력 사용.
> 쿨타임 기반 액티브 스킬.

**길들인 도깨비 리스트**

| 도깨비 | 해금 | 능력 | 쿨타임 |
|--------|------|------|--------|
| 먹보 | 관문 1 격파 | '탐식': 바닥패 1장 제거 (방해 카드 정리) | 3턴 |
| 장난꾸러기 | 관문 2 격파 | '속임수': 손패 1장과 바닥패 1장 교환 | 4턴 |
| 여우 | 관문 3 격파 | '환혹': 다음 매칭 시 와일드카드 1회 | 5턴 |
| 거울 | 관문 4 격파 | '반사': 보스 기믹 1회 반사 | 6턴 |
| 불꽃 | 관문 6 격파 | '소각': 바닥패 전체 리셋 (자발적) | 8턴 |
| 그림자 | 관문 8 격파 | '잠식': 보스 HP -15% (1판) | 10턴 |
| 뱃사공 | 시크릿 | '항해': 현재 턴 되감기 (Undo) | 12턴 |

**소환 슬롯:** 최대 2마리 장착 (영구 강화로 3마리까지)

**UI:**
```
[부적1][부적2][부적3]  [🔥먹보:2턴][🦊여우:준비]
```

**왜 차별적인가:**
- 카드 게임에 **액티브 스킬** 요소 추가 → 더 많은 전략적 선택
- 적이었던 보스를 길들이는 **감정적 보상**
- 부적(패시브) + 길들인 도깨비(액티브) = 이중 커스터마이징 레이어

### 2.5 사주팔자 (Destiny System) — 런마다 다른 운명

**컨셉:**
> 런 시작 시 **사주팔자**가 랜덤으로 결정.
> 한국 전통 사주(四柱) 시스템을 게임에 적용.
> 런 전체에 영향을 주는 "글로벌 모디파이어".

**사주 구성**
```
[년주] × [월주] × [일주] × [시주]
  │        │        │        │
 원소     기질     운세     흉길

예: 화(火) × 격(激) × 대길(大吉) × 살(殺)
→ "불의 격렬함, 크게 길하나 살기가 도사린다"
→ 칩 +30%, Go 배수 +1, but 보스 기믹 강화
```

**년주 (원소) — 기본 보너스**
| 원소 | 효과 |
|------|------|
| 목(木) | 띠 족보 칩 +20% |
| 화(火) | 광 족보 배수 +1 |
| 토(土) | 시작 엽전 +50 |
| 금(金) | 열끗 칩 +20% |
| 수(水) | 피 족보 활성 기준 -2 (8장부터) |

**월주 (기질) — 플레이 스타일 영향**
| 기질 | 효과 |
|------|------|
| 온(溫) | 매칭 실패 시 칩 +5 |
| 격(激) | Go 배수 +1 |
| 정(靜) | Stop 시 보너스 칩 +30 |
| 변(變) | 턴마다 30% 확률로 바닥패 1장 교체 |

**일주 (운세) — 보상 영향**
| 운세 | 효과 |
|------|------|
| 대길(大吉) | 보상 +50% |
| 길(吉) | 보상 +20% |
| 평(平) | 기본 |
| 흉(凶) | 보상 -20%, 저주 부적 등장 확률 증가 |
| 대흉(大凶) | 보상 -50%, but 클리어 시 넋 3배 |

**시주 (흉길) — 특수 변형**
| 흉길 | 효과 |
|------|------|
| 복(福) | 이벤트 +1회 |
| 록(祿) | 저승 장터 할인 -15% |
| 살(殺) | 보스 기믹 강화 |
| 파(破) | 랜덤 카드 2장 시작 시 제거 |
| 공(空) | 부적 슬롯 -1, but 카드 강화 2배 속도 |

**조합 예시**
```
수(水) + 정(靜) + 대길(大吉) + 복(福)
= "잔잔한 물의 대길한 복"
= 피 족보 활성 -2 + Stop 보너스 +30 + 보상 +50% + 이벤트 +1
→ 안정적인 피 빌드 + Stop 위주 플레이에 최적
→ "이번 런은 피 빌드로 안전하게 가자!"

화(火) + 격(激) + 대흉(大凶) + 살(殺)
= "격렬한 불의 대흉한 살기"
= 광 배수 +1 + Go 배수 +1 + 보상 -50% + 보스 강화
→ 극한 리스크 런 + 하지만 클리어 시 넋 3배
→ "미쳤지만 한 번 해볼까...?"
```

**왜 차별적인가:**
- 한국 전통 문화(사주팔자)를 게임 메커닉으로 변환
- 런마다 "오늘의 운세"처럼 다른 플레이 경험
- 같은 게임인데 5×4×5×5 = **500가지** 다른 런 조건
- 사주 조합 자체가 전략의 일부 ("이 사주면 피 빌드가 좋겠다")

### 2.6 먹물 연출 (Ink Dramatics) — 수묵화 이펙트 시스템

**컨셉:**
> 모든 핵심 순간을 **수묵화(水墨畫) 스타일**로 연출.
> 먹이 번지고, 붓이 그어지고, 서예가 나타나는 고유 비주얼 언어.

**연출 포인트**
| 순간 | 연출 |
|------|------|
| 족보 완성 | 화면에 먹물이 번지며 족보명이 서예체로 나타남 |
| Go 선택 | 붉은 먹물이 화면 가장자리에서 번짐 |
| 보스 등장 | 보스 실루엣이 먹물로 그려짐 → 색이 채워짐 |
| 카드 강화 | 카드에 금빛 먹물이 스며들며 문양 변화 |
| 크리티컬 매칭 | 붓이 화면을 가로지르는 일섬(一閃) 이펙트 |
| 최종 보스 | 화면 전체가 두루마리처럼 펼쳐지며 보스 등장 |
| 사망 | 먹물이 화면 전체를 삼킴 → 암전 |
| 승리 | 먹물이 걷히며 빛이 차오름 |

**왜 차별적인가:**
- 모든 카드 게임이 "카드가 빛남" 수준의 이펙트를 쓸 때
- 도깨비의 패는 **수묵화라는 고유한 비주얼 언어**를 가짐
- 한국적 정체성을 "기믹"이 아닌 **게임 경험의 DNA**로 내장
- 트레일러/GIF에서 즉시 구분 가능한 비주얼 아이덴티티
