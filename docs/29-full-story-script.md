# 도깨비의 패 — 전체 스토리 스크립트

> 저승에 끌려온 이름 없는 망자가 십왕의 심판을 거치며
> 자신의 죄를 깨닫고, 속죄와 해탈에 이르는 이야기.
>
> **이 게임에 마지막 단계는 없다.**
> 스토리는 윤회 1(관문 1~10)에서 펼쳐지지만,
> 게임은 윤회 2, 3, ... ∞로 영원히 계속된다.
> 매 윤회 10관문 클리어 시 "이승의 문"이 출현하며,
> 플레이어가 **원할 때** 문에 들어가면 엔딩을 볼 수 있다.
> 엔딩을 봐도 게임은 끝나지 않는다. 이어하기가 가능하다.
> **무한 구조에 대한 상세는 [30-infinite-game-structure.md](30-infinite-game-structure.md) 참조.**

## 프롤로그: 죽음

### 장면 P-1: 어둠

```
[완전한 어둠. 아무것도 없다.]

...

[먼 곳에서 물방울 떨어지는 소리]

...

[희미한 빛이 점처럼 나타난다]

목소리(내면): "...여기가 어디지?"

[빛이 서서히 커진다. 물 위에 떠 있는 자신을 발견한다.]
[검은 물. 수면에 비친 자신의 얼굴 — 하지만 뿌옇다.]

목소리(내면): "내 이름이... 뭐였지?"

[물속에서 손이 뻗어나와 발목을 잡는다]
[빨려들어간다]
```

### 장면 P-2: 삼도천

```
[눈을 뜬다. 강가.]
[검은 강물이 느리게 흐른다. 수면 위에 연꽃 등불.]
[하늘은 없다. 위도 아래도 끝없는 어둠.]
[다만 강 건너편에서 파란 불빛들이 명멸한다.]

[발소리. 삿갓을 쓴 인물이 다가온다.]
[긴 검은 두루마기. 얼굴은 삿갓 아래 그림자.]
[손에 긴 노(櫓)를 짚고 있다.]

뱃사공: "일어났군."

[잠시 침묵]

뱃사공: "네가 누군지 알고 싶겠지."
뱃사공: "나도 모른다. 네 이름은 이미 물에 씻겨갔으니."

[플레이어가 자신의 손을 본다. 반투명하다.]

뱃사공: "넌 죽었다."

[강 건너편에서 기괴한 웃음소리가 울려 퍼진다.]

뱃사공: "저승으로 건너가야 한다."
뱃사공: "열 분의 왕이 너를 심판할 것이다."
뱃사공: "하나하나 네 죄를 들춰내겠지."

[뱃사공이 품에서 낡은 화투 뭉치를 꺼낸다.]
[한지로 만든 패. 먹물 그림. 오래되었지만 도깨비불을 머금고 있다.]

뱃사공: "하지만... 방법이 있다."
뱃사공: "이 패로 왕들의 하수인, 도깨비를 이겨라."
뱃사공: "열 번 이기면 — 이승으로 돌아갈 수 있다."

[뱃사공이 패를 건넨다. 받는 순간 패에서 빛이 발한다.]

뱃사공: "물론, 지면..."

[뱃사공의 삿갓 아래에서 희미한 눈빛이 보인다.]

뱃사공: "영원히 이 강 밑에 가라앉는다."

[패를 쥔 손에 힘이 들어간다.]

뱃사공: "어떻게 하겠나?"

[선택지]
→ "치겠다." → (뱃사공 고개 끄덕) "좋은 눈이군."
→ "선택의 여지가 있나?" → (뱃사공 씁쓸한 미소) "...없지."
```

## Act 1: 각성 (관문 1~3)

### 관문 1: 귀시장 — 진광왕의 영역

#### 장면 1-1: 귀시장 진입

```
[안개를 헤치고 나아가면, 저승의 야시장이 펼쳐진다.]
[귀신들이 수상한 물건을 팔고 있다.]
[머리 없는 귀신이 떡을 팔고, 해골이 인형극을 한다.]
[파란 등불이 저승의 밤을 비춘다.]

뱃사공: "귀시장이다. 죽은 자들이 마지막 미련을 사고파는 곳."
뱃사공: "저기 보이나? 시장 한가운데."

[시장 중앙. 낡은 도박 탁자.]
[탁자 건너편에 거대한 그림자.]

뱃사공: "진광왕의 첫 번째 하수인이다."
뱃사공: "살생의 죄를 심판하는 왕의 앞잡이."
뱃사공: "조심해. 저 놈은 네 패를 먹어치운다."
```

#### 장면 1-2: 먹보 도깨비 등장

```
[그림자에서 거대한 형체가 모습을 드러낸다.]
[뚱뚱하고 붉은 피부. 입이 귀까지 찢어져 있다.]
[배에 화투 문양이 새겨져 있다. 침을 질질 흘리고 있다.]

먹보: "크하하하!"

[테이블이 흔들린다. 주변 귀신들이 슬금슬금 물러난다.]

먹보: "새로운 밥이 왔구나!"
먹보: "이승 냄새가 진하게 나! 오래간만이야!"

[먹보가 테이블에 양손을 짚는다. 쿵.]

먹보: "한 판 치자."
먹보: "이기면 보내준다. 약속하마."

[먹보가 입을 벌린다. 이빨 사이로 도깨비불이 새어나온다.]

먹보: "지면? 크크크..."
먹보: "맛있게 먹어주지."
```

#### 장면 1-3: 승리 후

```
[먹보가 테이블에 주저앉는다.]

먹보: "으억... 배가... 네 패가 체했어..."

[먹보의 몸에서 빛이 새어나온다.]
[먹보가 작아지기 시작한다. 거대했던 몸이 아이만 해진다.]

먹보(작아진): "에잇... 인정이다 인정..."
먹보(작아진): "네가 이겼으니 보내줄게."

[먹보가 주머니에서 무언가를 꺼내 건넨다.]

먹보(작아진): "이거... 가져가. 뭔가 쓸모 있을 거야."
→ [부적 또는 엽전 획득]

[먹보가 터벅터벅 어둠 속으로 사라지려 한다.]

뱃사공: "잠깐."

[뱃사공이 먹보를 본다.]

뱃사공: "데려가도 되겠나?"
먹보(작아진): "...마음대로 해."

→ [먹보 도깨비가 동료가 되었습니다]

[플레이어 내면의 목소리]
"살생... 내가 뭔가를 해쳤다고? 기억이... 조금..."

[순간 플래시백: 어두운 방. 깨진 유리. 비명.]
[금방 사라진다.]

뱃사공: "뭔가 떠올랐나?"
뱃사공: "앞으로 더 떠오를 것이다. 각오해라."
```

### 관문 2: 초강 — 초강왕의 영역

#### 장면 2-1: 얼어붙은 강

```
[귀시장을 빠져나오자, 세상이 달라졌다.]
[모든 것이 얼어있다. 강물도, 나무도, 공기조차.]
[강 위에 벌거벗은 혼령들이 얼음 속에 갇혀 있다.]
[그들의 입에서 안개가 새어나온다. 아직 의식이 있다.]

뱃사공: "초강이다. 도둑들이 벌을 받는 곳."
뱃사공: "훔친 것의 무게만큼 얼음이 두꺼워진다."

[플레이어가 얼음 위를 걷는다. 발밑에서 신음소리.]

뱃사공: "네 발 아래에도 도둑이 있다."
뱃사공: "너도... 무언가를 훔쳤었나?"

[플레이어 내면]
"훔쳤다? 내가? ...기억이 나지 않는다."
"하지만 이 추위가... 낯설지 않다."

[저 앞에 얼음으로 만든 도박 탁자가 보인다.]
[탁자 위에 서리가 내려앉아 있다.]
```

#### 장면 2-2: 얼음 도깨비

```
[얼음 기둥 뒤에서 도깨비가 나타난다.]
[날씬하고 창백한 피부. 온몸에 서리가 덮여있다.]
[입에서 나오는 숨이 즉시 얼어붙는다.]

얼음 도깨비: "..."

[말이 없다. 그냥 앉는다.]
[패를 꺼내 테이블에 놓는다. 패에 서리가 낀다.]

얼음 도깨비: "...치자."

[단 한마디. 차갑다.]

뱃사공: "말이 없는 놈이다. 하지만 방심하지 마."
뱃사공: "저 놈은 네 패를 얼린다."
```

#### 장면 2-3: 승리 후

```
[얼음 도깨비의 몸에서 얼음이 녹기 시작한다.]
[녹은 물이 눈물처럼 흐른다.]

얼음 도깨비: "...따뜻하군."

[처음으로 표정이 변한다. 조금 놀란 듯.]

얼음 도깨비: "오래간만에... 졌다."
얼음 도깨비: "가져가라. 내 안의 냉기를."
→ [얼음 도깨비 동료 해금 — 동결 면역]

[플래시백: 겨울. 누군가의 돈을 세고 있는 손. 떨리는 손.]
["미안해..." 라는 목소리가 울린다.]

뱃사공: "기억이 돌아오고 있군."
뱃사공: "네가 훔친 것... 돈이 아니었을지도 모른다."
```

### 관문 3: 환락루 — 송제왕의 영역

#### 장면 3-1: 환영의 누각

```
[얼음 세계가 끝나고, 갑자기 따뜻해진다.]
[화려한 누각이 나타난다. 꽃잎이 흩날린다.]
[기생들의 노래가 들린다. 아름다운 음악.]
[하지만 어딘가 불안하다. 꽃잎을 자세히 보면 — 피안화다.]

뱃사공: "환락루. 욕망에 빠진 자들이 영원히 춤추는 곳."
뱃사공: "여기서는 눈을 믿지 마라."
뱃사공: "보이는 것의 절반은 거짓이다."

[누각 안으로 들어간다.]
[화려한 방. 비단 쿠션. 술잔.]
[건너편에 아름다운 여인이 앉아 있다.]
[부채를 든 손이 우아하게 움직인다.]

[하지만 — 부채 뒤의 눈은 여우의 눈이다.]
```

#### 장면 3-2: 여우 도깨비

```
여우: "어머, 손님이시네."

[부채를 접는다. 미소가 요염하다.]

여우: "이승에서 오신 분은 오래간만이에요."
여우: "한 판... 놀아가실래요?"

[부채를 펼친다. 부채에 화투 문양이 그려져 있다.]

여우: "재밌을 거예요. 약속해요."

[부채 너머로 꼬리가 보인다. 하나... 둘... 아홉.]

뱃사공: (속삭임) "구미호다. 환영에 속지 마라."
뱃사공: "패 중에 가짜가 섞여있다. 진짜와 구분해야 한다."

여우: "뱃사공 아저씨~ 쓸데없는 말은 하지 마세요."
여우: (플레이어를 보며) "자, 시작하죠."
```

#### 장면 3-3: 승리 후

```
[여우의 미소가 사라진다.]
[부채가 바닥에 떨어진다.]
[아름다운 여인의 모습이 벗겨지고 — 작은 여우가 드러난다.]
[꼬리 9개. 은빛 털. 빛나는 눈.]

여우(원래 모습): "..."
여우: "나를 꿰뚫어보다니."
여우: "대단해요. 정말로."

[여우가 몸을 말며 작아진다.]

여우: "하나 알려줄게요."
여우: "당신이 이승에서 사랑했던 사람..."
여우: "아직 기다리고 있어요."

[플래시백: 햇빛이 드는 방. 웃고 있는 얼굴. 하지만 흐릿하다.]
["다녀올게." "조심해."]
[기억이 끊긴다.]

목소리(내면): "누구지... 이 사람은..."
목소리(내면): "내가... 돌아가야 할 이유가 있다."

→ [여우 도깨비 동료 해금]

뱃사공: "이름은 아직 모르겠나?"
뱃사공: "곧 알게 될 것이다. 거울 앞에 서면."

[뱃사공의 목소리가 평소보다 무겁다.]
```

## Act 2: 진실 (관문 4~6)

### 관문 4: 거울 궁전 — 오관왕의 영역

#### 장면 4-1: 진실의 거울

```
[무한한 거울이 있는 궁전.]
[모든 거울에 자신이 비친다 — 하지만 모두 다른 모습.]
[어떤 거울 속의 자신은 울고 있고, 어떤 거울 속은 웃고 있다.]
[어떤 거울 속의 자신은... 피투성이다.]

뱃사공: "오관왕의 궁전. 진실의 거울."
뱃사공: "이 거울은 거짓말을 비추지 않는다."
뱃사공: "네 안의 모든 진실을 보여줄 것이다."

[가장 큰 거울 앞에 서다.]
[거울 속에 비친 모습이 선명해진다.]

[거울 속의 자신이 입을 연다:]
"넌 도망쳤어."
"책임을 지지 않았어."
"그래서 여기 온 거야."

[거울이 깨진다. 파편에서 도깨비가 나타난다.]
[자신과 똑같이 생긴 도깨비. 거울 도깨비.]

거울 도깨비: "나는 네가 숨긴 진실이야."
거울 도깨비: "나를 이겨봐. 이길 수 있다면."
```

#### 장면 4-2: 승리 후 — 핵심 기억 회복

```
[거울 도깨비가 산산조각 난다.]
[파편들이 공중에 떠오르며 하나의 장면을 보여준다.]

[플래시백 — 선명한 기억:]
밤. 비가 온다.
차 안. 핸들을 잡은 손이 떨린다.
옆자리에 누군가. 웃고 있다.
"빨리 가자. 늦겠다."

전화가 울린다. 한 손으로 받는다.
잠깐 한눈을 판다.

빛.
충돌.
비명.

그리고 — 어둠.

[플래시백 종료]

[플레이어가 무릎을 꿇는다.]

목소리(내면): "나는..."
목소리(내면): "내가 운전을 했다."
목소리(내면): "내가... 그 사람을..."

뱃사공: "기억했군."
뱃사공: "네 죄는 부주의였다."
뱃사공: "사랑하는 사람을 다치게 했다."
뱃사공: "그리고 도망쳤지. 책임에서."

[긴 침묵]

뱃사공: "하지만 아직 끝이 아니다."
뱃사공: "앞으로 가면 더 많은 것을 마주하게 될 거다."
뱃사공: "그래도 갈 수 있겠나?"

[선택지]
→ "가겠다." → 뱃사공: "...그래."
→ "그 사람은 괜찮은 거야?" → 뱃사공: "...가면 알게 된다."
```

### 관문 5: 염라전 — 염라왕의 영역 (중간 결산)

#### 장면 5-1: 심판

```
[거대한 법정.]
[높은 곳에 염라왕이 앉아 있다.]
[옆에 업경대(業鏡臺) — 모든 행적을 비추는 거울.]

염라왕: "네가 기억했다고 들었다."

[업경대가 빛난다. 플레이어의 과거가 재생된다.]

[충격. 플레이어가 자살이었다는 것을 알게 된다.]

염라왕: "네가 사랑한 그 사람은 살았다."
염라왕: "하지만 너를 기다리고 있다."
염라왕: "아직도."

[긴 침묵]

염라왕: "심판을 내리겠다."
염라왕: "네 죄는 무겁지만, 속죄의 의지가 있다면..."
염라왕: "기회를 주마."

[염라왕이 손을 내밀며 화투패를 놓는다.]

염라왕: "나를 이겨라."
염라왕: "광 따위에 기대지 마라."
염라왕: "진짜 실력을 보여라."
```

## Act 3: 속죄 (관문 6~9)

### 관문 6: 화산 지옥 — 분노의 시험

```
[주인공이 자신에 대한 분노와 마주한다.]
"네가 화낸 건 세상이 아니라 너 자신이었다."

[화산 도깨비를 이기며 분노를 내려놓는다.]

승리 후:
"분노는 자격이 있다. 네가 잘못했으니까."
"하지만 분노만으로는 아무것도 바뀌지 않는다."
"내려놓아라."
```

### 관문 7: 황금 미궁 — 탐욕의 시험

```
[주인공이 '되돌리고 싶은 욕망'과 마주한다.]
"시간을 되돌릴 수 있다면 뭐든 하겠다고?"
"그 욕심이 너를 여기까지 끌고 온 거다."

[황금 도깨비를 이기며 집착을 내려놓기 시작한다.]

승리 후:
"과거는 바꿀 수 없다."
"할 수 있는 건 앞으로 가는 것뿐이다."
```

### 관문 8: 무한 회랑 — 나태의 시험

```
[끝없는 복도. 걸어도 걸어도 같은 풍경.]
"넌 이승에서도 도망만 했잖아."
"부주의했고, 도망쳤고, 멈춰섰고."
"이번에도 멈출 건가?"

[회랑 도깨비를 이기며 '행동하는 것'의 가치를 깨닫는다.]

승리 후:
"멈추지 않았군."
"그것만으로도 충분하다."
```

### 관문 9: 그림자 도시 — 시기의 시험

```
[도시가 나타난다. 하지만 모든 것이 주인공의 복제.]
[복제된 자신이 이승에서 행복하게 살고 있다.]
[사고도 없고, 도망도 없고, 죽음도 없는 자신.]

"부럽지? 이렇게 될 수 있었는데."
"하지만 이건 네가 아니야."
"네가 겪은 모든 것이 지금의 너를 만들었다."

[그림자 도깨비를 이기며 자기 자신을 받아들인다.]

승리 후:
"완벽하지 않아도 된다."
"너는 너다. 그것으로 충분하다."
```

## 엔딩

### 엔딩 A: 귀환 (트루 엔딩)
조건: 모든 기억 회복 + 모든 도깨비 동료 확보

```
→ "돌아가겠다. 이번에는 도망치지 않겠다."

뱃사공: "...그래. 그게 네 답이구나."
뱃사공: (미소) "잘 가라. 이번에는 제대로 살아."

[문을 통과한다.]
[밝은 빛.]

[눈을 뜬다. 병원. 하얀 천장.]
[옆에 누군가 있다. 손을 잡고 있다. 자고 있다.]
[그 사람의 얼굴이 — 처음으로 선명하게 보인다.]

"...나 돌아왔어."

[주머니에 화투패 한 장이 남아있다.]
[1월 송학 광. 한지 위에 학이 날고 있다.]
[패에서 미세한 도깨비불이 깜빡인다.]

[화면이 밝아지며 페이드아웃]

"도깨비의 패 — 끝"

[언락: 무한 모드 "황천의 끝"]
[언락: 윤회 시스템]
[언락: 뱃사공 동료]
```

### 엔딩 B: 잔류 (시크릿 엔딩)
조건: 특정 부적 "저승의 인연" 보유 + 뱃사공과 5회 이상 대화

```
→ "나도 이 강에 남겠다."

뱃사공: "...뭐?"

"도망쳐서 미안하다고만 하면 뭐가 달라져."
"여기서 다른 사람들을 도울래."
"너처럼."

뱃사공: "..."
뱃사공: "바보 같은 녀석."
뱃사공: (처음으로 웃는다) "...고맙다."

[뱃사공의 삿갓을 받는다.]
[삼도천 강가. 새로운 뱃사공이 서 있다.]
[강 건너에서 새로운 혼령이 눈을 뜬다.]

"어서 와. 넌 죽었어."
"하지만 방법이 있다."

[화면이 어두워지며]

"도깨비의 패 — 또 다른 시작"

[언락: 게임 시작 시 뱃사공 시점 모드]
[언락: 전용 부적 "뱃사공의 삿갓"]
```

### 엔딩 C: 윤회 (일반 엔딩)

"..."

"도깨비의 패 — 윤회"

[언락: 뉴게임+ (윤회 모드)]

## 숨겨진 이야기: 뱃사공의 과거

게임 곳곳에 숨겨진 단서를 모으면 뱃사공의 이야기가 밝혀진다.

```
뱃사공도 한때 이승의 사람이었다.
같은 죄 — 사랑하는 사람을 잃은 죄.

하지만 뱃사공은 "돌아가기"를 택하지 않았다.
"여기 남아서 속죄하겠다"고 했다.

수백 년 동안 삼도천에서 혼령들을 안내했다.
플레이어는 뱃사공이 기다려온 "마지막 혼령"이었다.

플레이어가 귀환하면 — 뱃사공도 드디어 쉴 수 있다.
플레이어가 잔류하면 — 뱃사공의 짐을 나눠 진다.
플레이어가 윤회하면 — 뱃사공은 계속 기다린다.

트루 엔딩 후 타이틀 화면:
삼도천 강가에 삿갓이 놓여있다. 뱃사공은 없다.
강물이 처음으로 맑게 흐른다.
```
