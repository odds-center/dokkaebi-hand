# 도깨비의 패 — Game Design Document (GDD)

## 1. 게임 아이덴티티

### 한 줄 요약
> "화투패를 모아 족보를 만들고, 족보로 보스 몬스터의 HP를 깎아 물리쳐라. 저승의 윤회는 끝이 없다."

### 엘리베이터 피치
도깨비의 패는 한국 전통 화투(고스톱)를 기반으로 한 **무한 로그라이트 보스 배틀 카드 게임**입니다.
플레이어는 저승에 끌려온 망자가 되어, 도깨비 보스들의 **HP를 족보 데미지로 깎아** 물리칩니다.
고스톱 전통 족보 + 섯다 2장 콤보 + 저승 고유 족보, **세 가지 족보 시스템**이 데미지의 원천입니다.
**10개 관문(Gate)으로 구성된 윤회(Cycle)**가 난이도를 높여가며 무한 반복됩니다.
보스는 항상 랜덤 — 랜덤 보스 + 랜덤 파츠 + 격노 페이즈로 매 전투가 다릅니다.
보스 파츠 랜덤화로 매 전투가 달라지는 전략적 깊이와 "한 판 더"의 중독성을 제공합니다.

### 레퍼런스 삼각형
```
        Balatro
       (칩×배수 = 데미지)
          /\
         /  \
        /    \
       /  도깨비의 패  \
      /________________\
Slay the Spire     고스톱+섯다(화투)
(보스 HP 전투)      (코어 룰셋 × 3종 족보)
```

### 차별점
| 요소 | Balatro | 도깨비의 패 |
|------|---------|------------|
| 카드 시스템 | 포커 (서양) | 화투 (한국) |
| 세계관 | 카지노/레트로 | 한국 저승/오컬트 |
| 족보 | 포커 핸드 5가지 | 3종 족보 (고스톱+섯다+저승) 30+ |
| 전투 시스템 | 목표 점수 달성 | 보스 HP를 족보 데미지로 깎기 |
| 리스크 시스템 | 없음 | Go/Stop (욕심 vs 안전) + 보스 반격 |
| 보스 기믹 | 블라인드 변형 | 도깨비별 고유 룰 파괴 + 격노 페이즈 |
| 문화적 정체성 | 범용 | 한국 민속/오컬트 |

## 2. 타겟 유저

### 1차 타겟: 카드 게임 마니아 (25~35세)
- Balatro, Slay the Spire, Inscryption 플레이 경험자
- "한 판만 더" 중독 루프에 빠지는 유저
- 수학적 최적화와 시너지 발견을 즐기는 유저

### 2차 타겟: 한국 문화 관심층
- 한국 전통 게임(화투/고스톱)에 친숙한 유저
- K-컬쳐(한국 민속, 도깨비, 저승)에 관심 있는 글로벌 유저
- 독특한 아트 스타일에 끌리는 인디 게임 팬

### 3차 타겟: 무한 진행 마니아
- Vampire Survivors, Hades 등 무한 스케일링 게임을 즐기는 유저
- 리더보드 경쟁과 극한 도전(높은 윤회 도달)을 즐기는 유저

### 유저 페르소나

**페르소나 A: "민수" (29세, 개발자)**
- Balatro 200시간 플레이, Slay the Spire 올클리어
- "새로운 카드 시스템이 신선하다. 화투 족보가 포커보다 복잡하고 재밌다"
- 시너지 조합 최적화에 몰두, 스프레드시트 분석 가능
- 세션당 30~60분

**페르소나 B: "유진" (33세, 디자이너)**
- 고스톱 명절에 가족과 즐김, 인디 게임 취향
- "한국적 세계관이 독특하다. 도깨비 디자인이 예쁘다"
- 스토리와 비주얼에 관심, 적당한 난이도 선호
- 세션당 15~30분

**페르소나 C: "Tom" (27세, 미국)**
- Inscryption으로 카드게임 입문, 한국 문화 관심
- "화투를 처음 접하지만 튜토리얼이 잘 되어있어서 배울 수 있다"
- 새로운 경험에 개방적, 로컬라이제이션 중요
- 세션당 20~40분

## 3. 핵심 경험 (Core Experience)

### 감정 곡선
```
긴장감 ▲
       │        ★ Go/Stop 결정 순간
       │       /│\
       │      / │ \     ★ 보스 기믹 발동
       │     /  │  \   /│
       │    /   │   \ / │  ★ 족보 완성!
       │   /    │    ╳  │ /
       │  /     │   / \ │/
       │ /      │  /   \│
       │/       │ /     │
       └────────────────────→ 시간
       패분배  중반   Go선택  결과
```

### 핵심 감정 순간 (Moments)

1. **"어? 이거 맞네?"** — 패 매칭 성공의 만족감
2. **"족보 완성이다!"** — 홍단/삼광/38광땡 등 족보 완성 시 쾌감
3. **"Go... 할까?"** — 리스크 선택의 긴장감과 손에 땀
4. **"HP 바가 쭉 깎인다!"** — 큰 데미지로 보스 HP를 한 방에 날리는 황홀함
5. **"보스가 격노했다!"** — HP 50% 이하에서 기믹 가속 + 반격 강화의 긴장
6. **"아... 먹혔다"** — 보스 기믹에 패를 빼앗기는 절망
7. **"이번엔 섯다 빌드로 가자"** — 새 런마다 다른 족보 전략 시도

### 세션 구조 — 무한 윤회 (Infinite Cycle)
```
[1회 윤회(Cycle): 30~50분]
├── 관문 1~3 (10분): 기본 학습, 첫 부적 획득, 보스 HP 전투 체험
├── 관문 4~6 (10분): 보스 HP 증가, 기믹 복합화, 격노 페이즈 체험
├── 관문 7~9 (10분): 부적 시너지 극대화, 섯다 족보 활용 필수
├── 관문 10 (5분): 윤회 최종 보스 — 해당 윤회의 정점
└── 다음 윤회 진입

[윤회 2+: 난이도 스케일링]
├── 보스 항상 랜덤 + 파츠 랜덤 재조합 → 매번 다른 전투
├── 보스 HP 증가, 반격 강화, 기믹 복합화
├── 강화 선택(3택 1)으로 빌드 성장 계속
└── 무한 반복 — 플레이어가 탈락할 때까지
```

## 4. 게임 필라 (Design Pillars)

### Pillar 1: 족보로 보스를 때린다 (Yokbo = Damage)
> 화투패를 모아 족보를 만들고, 족보가 곧 데미지. 보스 HP를 깎아라.

- 3종 족보(고스톱+섯다+저승) 동시 발동 → 데미지 극대화
- 보스 HP 바가 줄어드는 쾌감 = 핵심 보상
- 보스 격노(HP 50%) → 기믹 가속 + 반격 강화 → 긴장감 극대화
- **설계 원칙:** "어떤 족보로 얼마나 때릴 것인가?"가 모든 판의 핵심 질문

### Pillar 1.5: 욕심의 대가 (Risk & Reward)
> Go/Stop 시스템이 모든 판에 적용. 더 가져가려는 욕심이 최대의 무기이자 독.

- Go를 외칠 때마다 배수 폭발 but 패널티 누적 + 보스 반격 강화
- 3 Go 실패 = 즉사. 성공 = 전설 부적
- 보스 기믹이 Go 전략을 근본적으로 뒤흔듬
- **설계 원칙:** 모든 선택에 "이걸로 충분한가?"라는 질문을 내포

### Pillar 2: 한 판만 더 (Addictive Loop)
> 런마다 다른 부적 조합, 다른 전략, 다른 결과.

- 부적 시너지 발견의 즐거움 (20종 × 5슬롯 = 무한 조합)
- 상점에서의 빌드 분기 결정
- "이번엔 피 빌드로 해볼까" "광 빌드에 저승사자 명부를..."
- **설계 원칙:** 매 런이 새로운 퍼즐

### Pillar 3: 한국의 맛 (Cultural Identity)
> 화투의 아름다움과 한국 오컬트의 묘미를 현대적으로 재해석.

- 화투패의 시각적 아름다움 + 저승 테마 리디자인
- 도깨비 캐릭터의 개성과 대사
- 국악 퓨전 사운드
- **설계 원칙:** 한국인은 "익숙한데 새롭다", 외국인은 "낯설지만 매력적이다"

### Pillar 4: 깊이 있는 단순함 (Elegant Depth)
> 규칙은 3분에 배우고, 마스터는 100시간이 걸린다.

- 기본: "같은 월 맞추기" → 누구나 이해
- 중급: 족보 조합 전략 → 어떤 카드를 남기고 버릴 것인가
- 고급: 부적 시너지 + Go 타이밍 + 보스 기믹 대응 → 수학적 최적화
- **설계 원칙:** 쉬운 진입, 깊은 마스터리

## 5. 핵심 메커닉 요약

### 5.1 화투 매칭
- 48장 (12월 × 4장)
- 같은 월 카드를 매칭하여 획득
- 광/띠/열끗/피 4가지 타입

### 5.2 족보 시스템 (3종)
- **고스톱 전통:** 광 족보 (3광~5광), 띠 족보 (홍단/청단/초단), 개수 족보 (열끗 5+, 피 10+)
- **섯다 2장 콤보:** 38광땡, 땡, 알리, 독사, 구삥, 장삥, 장사, 세륙, 끗
- **저승 고유:** 저승꽃, 황천의 다리, 도깨비불, 삼도천, 사자의 노름
- 데미지 = 칩 × 배수 → 보스 HP 감소

### 5.3 Go/Stop
- 족보 완성 시 Go(계속) 또는 Stop(정산) 선택
- Go: 배수 증가 + 리스크 누적
- "욕심이 널 죽인다"

### 5.4 부적 (Talisman)
- 5슬롯 장착식 패시브/트리거 효과
- 시너지 조합이 핵심 전략
- 등급: 일반/희귀/전설/저주

### 5.5 보스전 (HP 전투)
- 보스는 HP 바를 가짐. 족보 데미지로 HP를 0으로 만들면 승리
- 보스는 항상 랜덤 (랜덤 보스 + 랜덤 파츠 + 랜덤 기믹 간격)
- 보스 파츠 시스템으로 랜덤 조합 (머리/몸통/특수)
- 격노 페이즈 (HP 50%): 기믹 가속 + 반격 강화
- 필살 페이즈 (HP 25%): 기믹 매 턴 + 반격 2배
- 라운드 사이에 보스 반격 (목숨/손패 감소)

### 5.6 무한 윤회 진행
- 10 관문 per 윤회, 윤회는 무한 반복
- 윤회마다 난이도 스케일링 (보스 HP 증가, 반격 강화, 기믹 복합화)
- 매 관문 클리어 시 강화 선택 (3택 1, Vampire Survivors 스타일)
- 통화: 엽전(냥) = 런 내, 넋(Soul) = 영구 강화
- 영구 진행 요소 (언락, 도감)

## 6. 경쟁작 분석

### Balatro
- **강점:** 중독성 있는 점수 시스템, 깔끔한 UI, 조커 시너지
- **약점:** 포커 이외의 문화적 깊이 부족, 보스전 단조로움
- **우리의 차별:** 화투 기반 + 보스 기믹 + Go/Stop 리스크

### Slay the Spire
- **강점:** 완성도 높은 로그라이크 구조, 밸런스
- **약점:** 카드 게임 자체보다 덱빌딩에 집중
- **우리의 차별:** 매 판의 Go/Stop 긴장감, 화투의 즉각적 피드백

### Inscryption
- **강점:** 스토리텔링, 메타 내러티브
- **약점:** 리플레이성 제한
- **우리의 차별:** 무한 리플레이 가능한 로그라이크 구조

### 한국 고스톱 앱들
- **강점:** 친숙한 룰
- **약점:** 혁신 없음, 도박/과금 위주
- **우리의 차별:** 로그라이크 + 부적 시너지 + 스토리
