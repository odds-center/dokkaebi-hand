# 확장 세계관 — 저승 십왕과 윤회의 화투판

## 1. 세계관 대전제

### 핵심 설정 변경
기존: 5층 → 염라대왕 격파 → 이승 귀환
**확장: 저승 십왕(十王) 체계 + 윤회 시스템 + 무한 황천**

> 한국/동아시아 저승 신화에는 **십왕(十王)**이 존재한다.
> 죽은 자는 10명의 왕 앞에서 심판을 받으며,
> 각 왕은 서로 다른 죄를 심판한다.

### 세계 구조
```
[이승] ←── 삼도천 ──→ [저승]
                         │
              ┌──────────┴──────────┐
              │                     │
         [본편: 십왕의 심판]     [무한: 황천의 끝]
         10개 관문 (스토리)     무한 웨이브 (엔드게임)
              │                     │
              └──────────┬──────────┘
                         │
                    [윤회의 문]
                  (뉴게임+, 영구 강화)
```

## 2. 저승 십왕 (十王) — 10개 관문

### 십왕 체계
한국 불교의 시왕(十王) 신앙을 기반으로 재해석.
각 왕은 특정 죄를 심판하며, 그에 맞는 **시련(기믹)**을 부여한다.

| 관문 | 왕 | 심판하는 죄 | 테마 | 환경 |
|------|-----|-----------|------|------|
| 1 | 진광왕(秦廣王) | 살생 | 귀시장 (Ghost Market) | 저승 입구, 야시장 |
| 2 | 초강왕(初江王) | 도둑질 | 초강(初江) | 얼어붙은 강, 벌거벗은 혼령 |
| 3 | 송제왕(宋帝王) | 음행 | 환락루(幻樂樓) | 요염한 환영의 누각 |
| 4 | 오관왕(五官王) | 거짓말 | 거울 궁전 | 진실을 비추는 무한 거울 |
| 5 | 염라왕(閻羅王) | 모든 죄 | 염라전(閻羅殿) | 거대 법정, 업경대 |
| 6 | 변성왕(變成王) | 분노 | 화산 지옥 | 용암, 불기둥, 유황 |
| 7 | 태산왕(泰山王) | 탐욕 | 황금 미궁 | 금은보화가 가득한 함정 미로 |
| 8 | 평등왕(平等王) | 나태 | 무한 회랑 | 끝없이 이어지는 복도 |
| 9 | 도시왕(都市王) | 시기 | 그림자 도시 | 상대가 나를 복제 |
| 10 | 오도전륜왕(五道轉輪王) | 집착 | 윤회의 문 | 모든 것이 뒤섞인 혼돈 |

### 관문별 상세

#### 관문 1: 귀시장 — 진광왕
```
환경: 저승 입구의 야시장. 귀신 상인들, 등불, 수상한 물건들.
분위기: 괴기하지만 활기. 첫 관문답게 안전하고 친절.

진광왕: 노인 모습, 수염이 길고 눈이 자비로움
"살생의 죄를 심판하마. 네가 이승에서 해친 것들의 원한을 받아라."

기믹: 매 라운드 시작 시, 바닥패 중 1장이 '원한 패'로 변함.
      원한 패를 매칭하면 추가 점수, 무시하면 턴 종료 시 -20 칩.

보스: 먹보 도깨비 (기존) + 진광왕 (2단계)
      진광왕은 원한 패를 3장으로 증가시킴.

라운드: 일반 3 → 보스 1 = 총 4라운드
```

#### 관문 2: 초강 — 초강왕
```
환경: 얼어붙은 삼도천 지류. 혼령들이 얼음 속에 갇혀있음.
분위기: 차갑고 고요. 바람 소리만 들림.

초강왕: 갑옷을 입은 무인, 차가운 표정
"도둑질한 자여, 네가 훔친 것들을 돌려놓아라."

기믹: '동결' 시스템. 매 2턴마다 손패 1장이 얼어붙음.
      얼어붙은 패는 2턴 후에야 사용 가능.
      해동 조건: 같은 월의 다른 패를 먼저 사용.

보스: 얼음 도깨비 → 초강왕 (2단계)
```

#### 관문 3: 환락루 — 송제왕
```
환경: 화려한 누각. 아름다운 환영이 유혹.
분위기: 퇴폐적이고 몽환적. 안개와 꽃잎.

송제왕: 아름다운 여성 모습, 부채 뒤에 해골 얼굴
"욕망에 빠진 자여, 환상과 진실을 구분해봐라."

기믹: '환영' 시스템. 바닥패 중 2장이 환영(가짜).
      환영 패는 매칭해도 사라지며 점수 없음.
      환영 구분법: 미세한 시각적 차이 (빛나는 테두리)
      → 주의력 테스트 + 리스크 관리

보스: 여우 도깨비(구미호) → 송제왕
```

#### 관문 4: 거울 궁전 — 오관왕
```
환경: 무한 거울이 있는 궁전. 진짜와 가짜 반전.
분위기: 현기증 나는 반사, 자아 분열.

오관왕: 사면에 눈이 달린 기괴한 존재
"거짓말쟁이여, 진실의 거울 앞에 서라."

기믹: '반전'. 매 라운드 시작 시 50% 확률로 점수 계산이 반전.
      광이 가장 낮은 점수, 피가 가장 높은 점수가 됨.
      반전 여부는 라운드 시작 시 공개.
      → 매 라운드 유연한 전략 전환 필요

보스: 거울 도깨비(도플갱어) → 오관왕
```

#### 관문 5: 염라전 — 염라왕 (중간 보스)
```
환경: 거대한 법정. 업경대(業鏡臺)가 모든 죄를 비춤.
분위기: 장엄하고 압도적. 중간 결산.

염라왕: 거대한 체구, 관복, 명부를 들고 있음
"모든 죄를 심판하마. 진정한 실력을 보여라."

기믹: '업경대'. 이전 4개 관문의 기믹이 랜덤 1개씩 적용.
      즉, 원한 패 + 동결 + 환영 + 반전 중 매 라운드 1개.
      → 모든 기믹 대응력 테스트

보스: 염라왕 (광 무효화 + 랜덤 기믹)
라운드: 일반 4 → 보스 2 = 총 6라운드

★ 중간 보스 격파 시 '윤회의 조각' 1개 획득
```

#### 관문 6~9: (간략)
```
6. 화산 지옥 (변성왕): '분노' 기믹 — 매칭 실패 시 보스 HP +10%
7. 황금 미궁 (태산왕): '탐욕' 기믹 — 상점 가격 2배, but 보상도 2배
8. 무한 회랑 (평등왕): '나태' 기믹 — 턴 제한 (5턴 내 클리어)
9. 그림자 도시 (도시왕): '복제' 기믹 — 보스가 플레이어 부적 복사하여 사용
```

#### 관문 10: 윤회의 문 — 오도전륜왕 (최종)
```
환경: 모든 것이 뒤섞인 혼돈의 공간. 이승과 저승의 경계.
분위기: 초현실적. 색이 뒤바뀌고, 중력이 바뀜.

오도전륜왕: 6개의 팔, 각 손에 다른 관문의 상징물
"집착을 버려라. 이승도 저승도 환상이다."

기믹: '혼돈'. 모든 이전 기믹이 매 턴마다 랜덤 적용.
      + 고유 기믹: '윤회'. 매 3턴마다 손패와 바닥패가 교환.

보스: 오도전륜왕 (3페이즈)
  Phase 1: 기본 (목표 500)
  Phase 2: 부적 전부 비활성화 (목표 400)
  Phase 3: 손패 5장 제한 + 모든 기믹 (목표 600)

★ 최종 보스 격파 → 스토리 엔딩 + '윤회의 문' 해금
```

## 3. 스토리 아크

### Act 1: 각성 (관문 1~3)
```
주인공은 자신이 왜 죽었는지 기억하지 못한다.
각 왕의 심판을 받으며 이승의 기억이 조금씩 돌아온다.

관문 1 후: "나는... 누군가를 해쳤었나?"
관문 2 후: "뭔가를 빼앗겼다... 아니, 빼앗았다?"
관문 3 후: "사랑했던 사람이 있었다... 이름이..."
```

### Act 2: 진실 (관문 4~6)
```
거울 궁전에서 자신의 죄를 직면한다.
주인공은 이승에서 큰 잘못을 저질렀음이 드러난다.
(구체적 죄는 의도적으로 모호하게 — 플레이어의 해석)

관문 4: 거울에 비친 과거의 자신
관문 5: 염라왕이 죄의 무게를 선언
관문 6: 분노와 자책 속에서 싸움
```

### Act 3: 속죄 (관문 7~9)
```
죄를 인정하고 속죄의 길을 걷는다.
이전에 적이었던 도깨비들이 조력자로 변하기도 한다.

관문 7: 탐욕을 버리는 시험
관문 8: 나태를 극복하는 시험
관문 9: 시기를 넘어서는 시험
```

### Act 4: 해탈 (관문 10)
```
최종 왕 앞에서 선택:
A) 이승으로 돌아간다 → 트루 엔딩 (조건: 모든 기억 회복)
B) 저승에 남아 도깨비가 된다 → 시크릿 엔딩 (조건: 특정 부적 보유)
C) 윤회를 받아들인다 → 일반 엔딩 (기본)

각 엔딩에 따라 다른 보상 언락.
```

## 4. NPC & 조력자 시스템

### 뱃사공 (메인 안내자)
- 관문 사이에서 등장하여 해설
- 각 왕에 대한 정보 제공
- 주인공의 과거를 간접적으로 암시
- **숨겨진 정체:** 주인공의 과거와 연결 (시크릿 엔딩 복선)

### 구제된 도깨비들
보스를 격파한 후, 일부 도깨비가 **동료**가 됨.
동료 도깨비는 **패시브 버프**를 제공 (영구).

| 도깨비 | 격파 조건 | 동료 버프 |
|--------|---------|----------|
| 먹보 도깨비 | 2턴 내 격파 | 매 라운드 시작 시 +1 손패 |
| 얼음 도깨비 | 동결 없이 격파 | 동결 면역 |
| 여우 도깨비 | 환영 0번 매칭 | 환영 감지 (시각적 힌트 강화) |
| 거울 도깨비 | 반전 라운드에서 승리 | 반전 시 배수 +1 |

## 5. 세계관 키워드 사전

| 용어 | 설명 |
|------|------|
| 삼도천(三途川) | 이승과 저승을 나누는 강. 건너면 돌아올 수 없다. |
| 업경대(業鏡臺) | 죽은 자의 모든 행적을 비추는 거울. 거짓말 불가. |
| 시왕(十王) | 저승을 다스리는 10명의 왕. 각각 다른 죄를 심판. |
| 엽전(葉錢) | 저승의 통화. 삼도천 뱃삯에서 유래. |
| 도깨비불(鬼火) | 저승에 떠다니는 인혼. 힘의 원천이자 위험. |
| 윤회(輪廻) | 생사의 순환. 뉴게임+의 세계관적 근거. |
| 피안화(彼岸花) | 저승에 피는 붉은 꽃. 기억과 망각의 상징. |
| 명부(冥簿) | 죽은 자의 이름이 적힌 장부. 이름이 지워지면 이승 귀환. |
| 황천(黃泉) | 저승의 깊은 곳. 끝이 없다. |
| 나락(奈落) | 가장 깊은 지옥. 되돌릴 수 없는 곳. |
