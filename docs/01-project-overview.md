# 도깨비의 패 (Dokkaebi's Hand) — 프로젝트 개요

## 컨셉
한국적 오컬트 세계관의 **무한 로그라이트 덱빌딩 카드 게임**.
**"화투패를 모아서 족보(고스톱+섯다+고유)를 만들고, 그 족보로 보스 몬스터의 HP를 깎아서 물리치는 게임"**
화투(Go-Stop) 룰 기반 + 섯다 2장 콤보 + 저승 고유 족보 + Vampire Survivors 스타일 무한 스케일링.

### 윤회 구조 (Infinite Cycle)
게임에 최종 스테이지는 없다. **10개 관문(Gate)으로 구성된 윤회(Cycle)**가 난이도를 높여가며 무한 반복된다.
각 관문마다 보스가 등장하며, **보스는 항상 랜덤** (랜덤 보스 + 랜덤 파츠 + 랜덤 기믹 간격).
족보를 완성하면 데미지(점×배)가 되어 보스의 HP를 깎고, 여러 라운드에 걸쳐 HP를 0으로 만들면 승리.
보스는 라운드 사이에 반격(counter-attack)을 가한다.

## 스토리
저승에 끌려온 망자가 도깨비들과 목숨을 건 화투판을 벌여 이승으로 돌아가는 이야기.

## 기술 스택
- **엔진:** Unity (C#)
- **렌더링:** 2D URP
- **타겟 플랫폼:** Steam PC (메인)

## 핵심 게임플레이 루프
```
[패 분배] → [턴: 손패 선택 → 바닥 매칭 → 뒤집기 매칭]
    → [족보 체크 (고스톱+섯다+고유)] → [Go/Stop 선택] → [데미지 정산]
    → [보스 HP 감소] → [보스 반격] → [HP 0 = 승리 / 플레이어 사망 = 패배]
```

## 전투 시스템 (족보 = 데미지)
- **데미지 = 칩(Chips) × 배수(Mult)** → 보스 HP에 적용
- 고스톱 족보 + 섯다 족보 + 저승 고유 족보 → 칩 + 배수 보너스
- 부적(Talisman) → 추가 칩/배수/특수효과
- Go 선택 → 배수 폭발 (리스크 증가, 보스 반격도 강화)

## 통화 시스템
- **엽전(냥):** 런 내 통화. 저승장터에서 부적/소모품 구매.
- **넋(Soul):** 영구 통화. 사망 시 획득, 영구 강화(윤회 강화)에 사용.
