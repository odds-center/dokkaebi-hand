| 청단 | Blue ribbon | Blue Scroll | 青短 |

**도깨비 번역**
- 도깨비는 번역하지 않고 **Dokkaebi** 유지 (K-문화 아이덴티티)

- 염라대왕: "King Yeomra" (번역) / "閻魔大王" (일본어)

### 3.2 텍스트 길이 대응
```
한국어 기준 → 영어는 약 1.3~1.8배 길어짐
→ UI 레이아웃에 텍스트 오버플로우 대비 필요
→ 부적 설명: 최대 2줄 (한국어 기준 30자)
→ 버튼 텍스트: 최대 8자 (한국어 기준)
```

### 3.3 보이스 로컬라이제이션
- **v1.0:** 한국어 보이스 + 전 언어 자막
- **업데이트:** 일본어 보이스 추가 고려
- 영어 보이스는 비용 대비 효과 검토 후 결정

## 4. 기술적 구현

### 로컬라이제이션 시스템
```csharp
// Unity Localization Package 사용
// Assets/Localization/StringTables/

GameStrings_ko.asset  // 한국어 (기본)
GameStrings_en.asset  // 영어
GameStrings_ja.asset  // 일본어
```

### 키 네이밍 규칙
```
UI_BUTTON_GO          → "Go" / "고"
UI_BUTTON_STOP        → "Stop" / "스톱"
UI_LABEL_CHIPS        → "Chips" / "칩"
UI_LABEL_MULT         → "Mult" / "배수"

CARD_NAME_JAN_GWANG   → "Pine Crane" / "송학 광"
CARD_NAME_JAN_HONGDAN → "Pine Red Scroll" / "송학 홍단"

TALISMAN_BLOOD_OATH_NAME → "Blood Oath" / "피의 맹세"
TALISMAN_BLOOD_OATH_DESC → "Each Pi card provides +1 Mult" / "피 패 1장당 +1 배수"

BOSS_GLUTTON_INTRO    → "Your cards smell delicious!" / "네 패에서 맛있는 냄새가 나는구나!"
BOSS_GLUTTON_DEFEAT   → "Ugh... too full..." / "으억... 배가 너무 불러..."

EVENT_WANDERER_DESC   → "You find a lost soul..." / "길을 잃은 망자를 발견한다..."
EVENT_WANDERER_OPT_A  → "Help them" / "도와준다"
```

### 폰트 대응
| 언어 | 폰트 요구사항 |
|------|-------------|
| 한국어 | 한글 전체 (11,172자) + 한자 기본 |
| 영어 | 라틴 기본 |
| 일본어 | 히라가나 + 가타카나 + 한자 (JIS 1수준) |

**폰트 선택 기준:**
- 먹물/전통 느낌의 서체
- 한/영/일 통합 지원 또는 언어별 개별 폰트
- 라이선스: 상업용 무료 or 구매

## 5. 글로벌 유저를 위한 화투 소개

### 인게임 "화투란?" 섹션
```
[메인 메뉴 → 도감 → 화투 소개]

"화투(花札)는 한국의 전통 카드 게임입니다.
12개의 월(Month)을 상징하는 48장의 카드로 구성되어 있으며,
각 월에는 4장의 카드가 있습니다.

같은 월의 카드를 매칭하여 모으고,
특정 조합(족보)을 완성하면 점수를 얻습니다.

도깨비의 패에서는 이 전통적인 규칙에
부적(Talisman)과 Go/Stop 시스템을 더해
전략적 깊이를 극대화했습니다."
```

### 각 월 소개 (도감)
```
1월 — 송학 (Pine & Crane)
"새해의 시작을 상징하는 소나무와 학.
한국에서 소나무와 학은 장수와 고결함을 상징합니다."

[카드 4장 이미지]
광: 소나무 위의 학 (20점)
홍단: 붉은 시 리본 (10점)
피: 소나무 잎 (1점) ×2
```
