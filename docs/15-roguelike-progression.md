# 로그라이크 진행 & 메타 시스템 설계

## 1. 무한 윤회(Infinite Reincarnation) 구조

> 게임에 최종 스테이지는 없다. 로그라이크 + Vampire Survivors 스타일의 **무한 진행** 구조.

### 윤회(Reincarnation) 기본 개념
- **1 윤회 = 10개 관문(Gate)**
- 윤회는 무한 반복되며, 회차마다 난이도가 상승
- 각 윤회 끝에서 **"이승의 문(Gate to the Living)"** 선택지 등장
  - 진입 → 엔딩 감상 → 게임 계속 (다음 윤회)
  - 거부 → 바로 다음 윤회 진입
- 엔딩은 **선택사항**, 게임 자체는 플레이어가 탈락할 때까지 계속

### 단일 윤회 플로우
```
[윤회 시작]
  │
  ├─ 튜토리얼 (첫 윤회만)
  │
  ├─ 관문 1: 보스전 (3판) → [저승 장터] → [이벤트]
  ├─ 관문 2: 보스전 (3판) → [저승 장터] → [이벤트]
  ├─ 관문 3: 보스전 (3판) → [저승 장터]
  ├─ 관문 4~9: 보스전 + 저승 장터 + 이벤트 (난이도 점진 상승)
  │
  └─ 관문 10: 윤회 보스 (5판) → [이승의 문 선택]
       ├─ "이승의 문 진입" → 엔딩 감상 → 다음 윤회
       └─ "거부" → 바로 다음 윤회

예상 플레이 시간: 윤회 1개당 30~50분
윤회 2+: 보스 파츠 랜덤화로 매번 다른 전투
```

### 윤회별 자원 흐름
```
윤회 1 시작: 목숨 3, 엽전 100, 부적 0

관문 1~3 클리어: 목숨 3, 부적 1~2, 빌드 방향 설정
   → 저승 장터/이벤트에서 초기 빌드 확보
   → 매 관문 클리어 시 강화 선택 (3택 1)

관문 4~6 클리어: 부적 3~4, 강화 누적 → 시너지 발현
   → 중간 난이도 구간, 빌드 최적화

관문 7~9 클리어: 부적 4~5, 강화 다수 → 빌드 완성
   → 고난이도 구간, Go/Stop 전략 중요

관문 10 윤회 보스: 모든 자원 투입
   → 클리어 시: "이승의 문" 선택 → 다음 윤회 (난이도 상승, 자원 일부 유지)
```

## 2. 관문 내 맵 시스템 (Alpha 확장)

### 분기 맵 (Slay the Spire 스타일)
```
MVP에서는 선형 구조. Alpha에서 각 관문 내 분기 맵 도입.

[관문 시작]
  │
  ├── 전투 ──┐
  │              ├── 저승 장터 ──┐
  ├── 이벤트 ─┘                   ├── 전투 ──→ [관문 보스]
  │                                  │
  └── 엘리트 ────────────────────┘

노드 타입:
전투: 잡졸 도깨비와 1판 (보스보다 쉬움)
이벤트: 랜덤 이벤트
저승 장터: 부적/소모품 구매
엘리트: 강화 도깨비 (높은 보상)
휴식: 체력 회복 or 부적 강화
보스: 관문 보스전
```

### 분기 선택의 의미
- **전투 루트:** 더 많은 판 = 더 많은 강화 기회
- **이벤트 루트:** 자원 획득 기회 but 불확실
- **엘리트 루트:** 고위험 고보상 (희귀/전설 부적)
- **플레이어의 현재 상태에 따라 최적 루트가 달라짐**
- **윤회 후반 관문:** 엘리트 비율 증가, 휴식 노드 감소

## 3. 메타 진행 (영구 언락)

### 3.1 도감 시스템
- 획득한 모든 카드/부적/보스 기록
- 각 부적별 사용 통계 (장착 횟수, 최고 점수 기여)
- 각 보스별 클리어 통계

### 3.2 업적 시스템
| 업적 | 조건 | 보상 |
|------|------|------|
| 첫 승리 | 관문 1 클리어 | 부적 "초심자의 부적" 언락 |
| 오광 달성 | 한 판에서 오광 완성 | 부적 "광기의 광" 언락 |
| 3 Go 생존 | 3 Go 성공 | 부적 "죽음을 이긴 자" 언락 |
| 피의 왕 | 피 15장 이상 획득 | 부적 "피의 왕관" 언락 |
| 윤회 완주 | 윤회 1 클리어 (관문 10) | 난이도 "저승+" 언락 |
| 무부적 클리어 | 부적 없이 관문 5 이상 | 부적 "무의 경지" 언락 |
| 부자 | 엽전 500 이상 보유 | 저승 장터에 전설 부적 등장률 증가 |
| 전족보 | 한 윤회에서 모든 족보 완성 | 스킨 "금빛 화투" 언락 |
| 무한 탐험가 | 윤회 5 도달 | 부적 "윤회의 고리" 언락 |
| 황천의 끝 | 윤회 10 도달 | 시크릿 엔딩 해금 |

### 3.3 난이도 시스템
| 난이도 | 해금 조건 | 변경 사항 |
|--------|---------|----------|
| 이승 (Easy) | 기본 | 보스 HP -20%, 목숨 4개 |
| 저승 (Normal) | 기본 | 기본 밸런스 |
| 저승+ (Hard) | 염라 격파 | 보스 HP +30%, 목숨 2개, 저승 장터 가격 +50% |
| 나락 (Nightmare) | 저승+ 클리어 | 보스 HP +50%, 목숨 1개, 보스 기믹 강화 |

### 3.4 시작 부적 선택 (언락 기반)
```
[새 게임 시작]
"시작 부적을 선택하세요" (최대 1개)

┌──────────┐ ┌──────────┐ ┌──────────┐
│ 피의 맹세 │ │ 홍살문   │ │ ???      │
│ (기본)   │ │ (언락)   │ │ (미해금)  │
└──────────┘ └──────────┘ └──────────┘

→ 첫 런은 선택 없이 시작
→ 업적 달성할수록 시작 부적 옵션 증가
→ 시작 부적은 런의 초기 방향을 결정
```

## 4. 잡졸 전투 (Alpha 확장)

### 잡졸 도깨비
보스 사이에 등장하는 가벼운 전투. 1판으로 끝남.

| 잡졸 | HP | 특징 |
|------|-----|------|
| 꼬마 도깨비 | 100 | 기믹 없음 (연습용) |
| 술주정 도깨비 | 120 | 랜덤으로 바닥패 2장 교체 |
| 쇠뿔 도깨비 | 150 | 열끗 데미지 무효화 |
| 외눈 도깨비 | 130 | 손패 2장이 뒤집힘 (기억력) |
| 절름발이 도깨비 | 140 | 3턴 제한 (빠른 클리어 필요) |

### 엘리트 도깨비
잡졸보다 강력, 보스보다 약함. 높은 보상.

| 엘리트 | HP | 특징 | 보상 |
|--------|-----|------|------|
| 쌍둥이 도깨비 | 200 | 기믹 2개 동시 적용 | 희귀 부적 |
| 늙은 도깨비 | 250 | 띠/피만 유효 | 전설 부적 (낮은 확률) |
| 도깨비 왕자 | 220 | 2턴마다 HP +20 | 엽전 150 |

## 5. 일일 도전 (Beta에서 추가)

### 컨셉
- 매일 다른 시드로 동일한 런 제공
- 전 세계 유저가 같은 조건에서 경쟁
- 리더보드: 최고 점수 기준 순위

### 일일 도전 변형
| 변형 | 설명 |
|------|------|
| 부적 천국 | 시작 부적 3개, 저승 장터에서 부적 50% 할인 |
| 광의 시대 | 광 패 2배 등장, 광 점수 2배 |
| 피의 저주 | 모든 피 패가 쌍피, but 매턴 피 1장 소멸 |
| Go or Die | 목숨 1개, Go 배수 2배 |
| 가난한 망자 | 엽전 0, 저승 장터 이용 불가, 이벤트만으로 빌드 |

## 6. 리플레이 가치 설계

### "한 판 더"의 동기
1. **새로운 부적 조합:** "이번엔 피 빌드에 저승사자 명부를..."
2. **업적 도전:** "무부적 클리어를 해봐야지"
3. **높은 난이도:** "나락 난이도에서 염라를 이기겠어"
4. **일일 도전:** "오늘 리더보드 1위를 노려볼까"
5. **도감 완성:** "아직 안 써본 부적이 있다"
6. **운의 변수:** "이번엔 좋은 부적이 나올지도..."

### 예상 콘텐츠 소비 시간
| 콘텐츠 | 시간 |
|--------|------|
| 윤회 1 클리어 (이승 난이도) | 2~4시간 |
| 저승 난이도 윤회 1 클리어 | 5~10시간 |
| 윤회 5 도달 | 15~25시간 |
| 전 업적 달성 | 30~50시간 |
| 윤회 10+ (극한 도전) | 50~80시간 |
| 전 도감 완성 | 60~100시간 |
| 무한 윤회 경쟁 + 리더보드 | 무한 |
