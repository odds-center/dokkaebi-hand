# Go/Stop 룰 및 리스크 시스템

> **핵심:** Go-Stop 매칭은 카드를 모으는 수단이고, 모은 카드로 족보를 만들고, 족보 = 데미지로 보스 HP를 깎는다.

## 기본 Go-Stop 진행
1. **패 분배:** 플레이어 손패 10장, 바닥패 8장, 나머지 뽑기패
2. **턴 진행:**
   - 손패에서 1장 선택 → 바닥과 같은 월 매칭 → 카드 획득
   - 뽑기패에서 1장 뒤집기 → 바닥과 같은 월 매칭 → 카드 획득
3. **족보 완성 시:** Go 또는 Stop 선택
4. **Stop:** 현재 데미지로 보스 HP 감소 → 보스 반격 → 다음 라운드
5. **Go:** 계속 진행, 배수 증가 + 리스크 증가 (더 큰 데미지 가능)

## 매칭 규칙
```
같은 월 0장 → 바닥에 놓기
같은 월 1장 → 둘 다 획득
같은 월 2장 → 하나 선택 후 획득
같은 월 3장 → 전부 획득 (쓸)
```

## Go/Stop 리스크 상세

### Go를 선택하면:
- 현재 족보/데미지 유지한 채 게임 계속
- 더 많은 족보를 모아 데미지 극대화 가능 (보스 HP를 더 깎을 수 있음)
- 단, 리스크 누적 (보스 반격도 강화될 수 있음)

### 리스크 테이블
| Go 횟수 | 최종 배수 | 다음 라운드 영향 | 실패 시 |
|---------|----------|----------------|---------|
| 1회 | ×2 | 보스 반격 1.5배 | 목숨 -1 |
| 2회 | ×4 | 손패 -1장 | 목숨 -1 |
| 3회+ | ×10 | - | **즉사** (목숨 전부 소멸) |

### 3 Go 성공 시 보너스
- 최종 배수 x10
- **전설 부적 1개 드랍 확정**
- 극히 높은 리스크 but 최대 리턴

## 전략적 고려사항
- 1 Go는 거의 항상 이득 (배수 2배 → 보스 HP를 더 깎음)
- 2 Go는 현재 족보가 강할 때만 (손패 감소 패널티)
- 3 Go는 "마지막 한 수" — 목숨 3개 상태에서만 시도 권장
- 보스 HP가 많이 남았을 때는 Go로 큰 데미지를 노리는 것이 유리
- 보스 격노 페이즈(HP 50% 이하)에서는 반격이 강해지므로 빠른 Stop도 고려
- 보스 기믹에 따라 Go 전략이 크게 달라짐
  - 먹보 도깨비: Go할수록 손패 손실 가속
  - 염라대왕: 광 없이 데미지 쌓기 어려우므로 섯다 족보 + Go 필수

## 데미지 계산 공식
```
데미지 = (기본 칩 + 고스톱 족보 칩 + 섯다 족보 칩 + 저승 족보 칩 + 부적 칩)
       × (기본 배수 + 고스톱 족보 배수 + 섯다 족보 배수 + 저승 족보 배수 + 부적 배수)
       × Go 배수

데미지 → 보스 HP에서 차감
보스 HP = 0 → 승리
```
