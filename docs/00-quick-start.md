# 도깨비의 패 — Quick Start

## Unity에서 실행하기

### 1. 요구 사항
- **Unity 2022.3 LTS** 이상 (2023.x도 가능)
- TextMeshPro 패키지 (자동 설치됨)

### 2. 프로젝트 열기
1. Unity Hub에서 "Open" → `dokkaebi-hand` 폴더 선택
2. Unity가 패키지를 임포트할 때까지 대기
3. TextMeshPro 초기 설정 팝업이 나오면 "Import TMP Essentials" 클릭

### 3. 씬 설정
1. `File > New Scene > Basic (Built-in)` 으로 빈 씬 생성
2. Hierarchy에서 빈 GameObject 생성 → 이름: `GameBootstrap`
3. Inspector에서 `Add Component > GameBootstrap`
4. `useMockup` 체크 확인 (기본 true)
5. `File > Save Scene As` → `Assets/Scenes/MainScene.unity`

### 4. 플레이!
- 상단 ▶ 버튼 클릭
- 마우스/터치로 모든 조작

### 5. 게임 흐름
```
메인 메뉴 → 새 게임
  → 사주팔자 생성
  → 도깨비의 거래 (축복 선택)
  → 1번째 관문:
    [섯다 승부] 2장 대결 → 버프/디버프
    [고스톱] 패 매칭 → 족보 빌드 (시너지)
    [고/스톱] 고=배수↑위험↑ / 스톱=공격 페이즈
    [공격] 모은 패 중 2장 섯다 공격 → 보스 HP↓
    [반격] 보스 기믹 + 카운터
    → 보스 HP 0 = 격파!
  → 강화 선택 (3택 1)
  → 저승 장터 (부적/소모품/대장간)
  → 다음 관문...
  → 10관문 → 이승의 문 → 계속 or 엔딩
```

### 6. 조작법
- **손패 클릭**: 바닥에 카드 내기 (매칭)
- **고/스톱 버튼**: 족보 완성 시 선택
- **공격 페이즈**: 먹은 패 중 2장 클릭 → [공격!] 버튼
- **상점**: 아이템 클릭으로 구매
- **동료 도깨비**: 하단 아이콘 클릭으로 스킬 사용

### 7. 핵심 용어
| 용어 | 의미 |
|------|------|
| 판 | 라운드 (고스톱 한 판) |
| 관문 | 보스가 지키는 저승의 문 |
| 윤회 | 10관문 = 1윤회, 무한 반복 |
| 넋 | 영구 강화 화폐 |
| 엽전(냥) | 런 내 화폐 |
| 고/스톱 | Go/Stop (고스톱 원래 용어) |
| 먹다/쓸/쪽 | 매칭 결과 (고스톱 용어) |
| 시너지 | 고스톱 족보가 섯다 공격을 강화하는 효과 |
