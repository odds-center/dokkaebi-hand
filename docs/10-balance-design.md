## 1. 데미지 경제 (Damage Economy)

> **핵심:** 족보 = 데미지. (섯다 기본 + 칩) × 배수 × Go 배수 = 보스 HP에서 차감.

### 기본 수치 체계
```
최종 데미지 = (섯다 기본 데미지 + 누적 칩 + 부적 칩) × 누적 배수 × 부적 배수 × Go 배수

칩: 족보 완성으로 누적하는 기본 데미지 수치 (덧셈)
배수: 족보로 증가하는 데미지 승수 (곱셈, 상한 10.0)
Go 배수: Go 1=×2, Go 2=×3, Go 3=×8

최소 보장 데미지: 15 (족보 0이어도 진행 가능)
```

### 복합 족보 시나리오 (데미지 → 보스 HP)

**시나리오 A: 관문 1 보스 (HP 300)**
```
홍단 완성: 칩 120, 배수 3 (1+2)
데미지: 120 × 3 = 360 → 보스 HP 300 - 360 = 0 ✓ (1라운드 킬)
```

**시나리오 B: 관문 5 보스 (HP 1500)**
```
라운드 1: 삼광 + 청단: 칩 270, 배수 5 → 270 × 5 = 1350 데미지
보스 반격: 목숨 -1
라운드 2: 초단만: 칩 120, 배수 3 → 120 × 3 = 360 데미지
총 데미지: 1350 + 360 = 1710 > 1500 ✓ (2라운드 킬)
```

**시나리오 C: 고스톱 + 섯다 복합 폭발**
```
오광 + 홍단 + 38광땡 + 피 12장:
고스톱 칩: 300+120+50 = 470, 섯다 칩: +400 = 870
배수: 1+5+2+8 = 16
3 Go: 배수 ×10 = 160
피의 맹세 (피 12장): 배수 +12 = 172
저승사자 명부 (끝자리 4): ×4 = 688
데미지: 870 × 688 = 598,560
```

**시나리오 D: 섯다 중심 빌드**
```
알리(1+2월) + 홍단(1,2,3월 띠):
섯다 칩 180 + 고스톱 칩 120 = 300, 배수 1+3+2 = 6
데미지: 300 × 6 = 1800 (섯다+고스톱 시너지)
```

## 2. 보스 HP 밸런스

### 설계 원칙 (v4 패치)
```
보스 HP = target_score × 1.4^(윤회번호 - 1)
관문 내 스케일링: target × (1.0 + 0.05 × (관문번호 - 1))

20관문 = 1윤회
```

### 보스 target_score 범위

| 티어 | target_score | 윤회1 HP 범위 | 설계 의도 |
|------|-------------|-------------|----------|
| 입문 (normal) | 150~230 | 150~230 | 4~6판이면 격파 |
| 초급 (normal) | 230~280 | 230~280 | 족보 1~2개 필요 |
| 중급 (elite) | 320~500 | 320~500 | 복합 족보 + Go 필요 |
| 보스 (boss) | 600 | 600 | 윤회 마지막 관문 |
| 재앙 (calamity) | 2500~4000 | 2500~4000 | 극한 도전 |

### 보스 HP 검증 시나리오 (v4 시뮬레이션)

**입문 관문 (HP 150) — 초보도 가능**
```
최악(족보 0): 7판에 격파 (최소 데미지 15 보장)
김초보(홍단 1개): 4판 격파 ✓
이전략(홍단+Go1): 2판 킬! ✓
```

**관문 20 염라대왕 (HP 600) — 전략 필요**
```
이전략(홍단+초단+Go1): 5판 격파 ✓
Go 2 이상 활용하면 3~4판 가능
```

### 염라대왕 밸런스 검증 (광 무효, HP 500)
```
광 없이 HP 500 깎기:

Plan A: 섯다 족보 활용 (신규 전략!)
- 알리(180칩, +3배) → 180 × 4 = 720 데미지 → 1라운드 킬 가능!
- 섯다 족보가 광 무효를 우회하는 핵심 전략

Plan B: 홍단 + 피 빌드
- 홍단(120칩, +2배) + 피 12장(50칩)
- 170 × 3 = 510 데미지 → 1라운드 킬 ✓

Plan C: 2라운드 분할
- 라운드1: 초단 120 × 3 = 360, 라운드2: 피만 170 → 총 530 ✓
- 반격 1회 감수

Plan D: 섯다 + 고스톱 복합
- 장삥(150칩, +2배) + 초단(120칩, +2배)
- 270 × 5 = 1350 데미지 → 여유 클리어 ✓
```

**결론:** 염라대왕전은 섯다 족보가 광 무효를 우회하는 핵심 전략. 복합 족보 빌드 유도.

## 3. 부적 밸런스

### 효과 등급별 수치 가이드라인

| 등급 | 칩 추가 | 배수 추가 | 배수 승산 | 특수 효과 |
|------|---------|----------|----------|----------|
| 일반 | +10~30 | +1~2 | - | 약한 조건부 |
| 희귀 | +30~50 | +2~4 | x1.5 | 중간 조건부 |
| 전설 | +50~100 | +4~8 | x2~4 | 강력한 조건부 |
| 저주 | 0 | -1~-3 | - | 부정적 효과 |

### MVP 부적 밸런스 분석

**피의 맹세 (일반) — 피당 +1 배수**
```
최선: 피 15장 = +15 배수 → 매우 강력
최악: 피 3장 = +3 배수 → 적당
평균: 피 8장 = +8 배수 → 강력

판정: 일반 등급치고 강함. 피 빌드 핵심.
조정 검토: 피 10장 이상일 때만 발동? or +0.5 배수?
→ 현재 유지 (피 빌드의 핵심 보상으로)
```

**도깨비 감투 (희귀) — 띠당 보스 HP -5%**
```
최선: 띠 9장 = -45% → 보스 HP 200 → 110
최악: 띠 1장 = -5% → 미미
평균: 띠 4장 = -20% → 보스 HP 200 → 160

판정: 적절. 띠를 모을수록 클리어가 쉬워지는 선순환.
주의: 홍단+청단+초단 = 띠 9장 → -45%는 과할 수 있음
조정 검토: 최대 -30% 캡? → Alpha에서 테스트
```

**저승사자의 명부 (전설) — 끝자리 4일 때 x4**
```
발동 확률: 약 10% (끝자리 0~9 중 4)
최선: 300 × 10 → 3000, 끝자리 0 ✗ / 조정하여 끝자리 4 → x4 = 12000
최악: 발동 안 됨 → 효과 없음

판정: 하이리스크 하이리턴. 전설 등급에 적합.
전략: 다른 부적으로 점수 끝자리를 4로 맞추는 고급 플레이
```

### 부적 시너지 폭발 상한선 테스트

**최대 데미지 시나리오 (5슬롯 풀장착 + 3종 족보)**
```
고스톱: 오광(300칩/+5배) + 홍단(120칩/+2배) + 피12장(50칩)
섯다: 38광땡(400칩/+8배)
저승: 도깨비불(300칩/+7배)
기본: 1170칩 × 23배수 = 26,910 데미지

부적 적용:
1. 피의 맹세: +12 배수 → 1170 × 35 = 40,950
2. 홍살문: +30칩 → 1200 × 35 = 42,000
3. 도깨비 감투: 보스 HP 감소 (데미지에 직접 영향 없음)
4. 광기의 광: 변이 효과 (수치 변동)
5. 저승사자 명부: 끝자리 4 시 ×4 → 1200 × 140 = 168,000

3 Go: ×10 → 1200 × 1400 = 1,680,000 데미지
```

**상한선 판정:** 약 168만 데미지. 현실적 달성 확률은 극히 낮음 (3종 족보 올콤보+끝자리4+3Go 동시). 고윤회 보스 HP가 1만+ 이므로 밸런스 위험 없음.

## 4. Go/Stop 기대값 분석

### Go 배수 테이블 (v4 패치)

| Go 횟수 | 배수 | 리스크 | 판정 |
|---------|------|--------|------|
| Go 0 (스톱) | ×1 | 없음 | 안전 |
| Go 1 | **×2** | 다음 라운드 난이도↑ | 거의 항상 이득 |
| Go 2 | **×3** | 손패 -1 | 상황에 따라 |
| Go 3 | **×8** | 즉사 10% | 숫자 폭발! 극한 도박 |

### Go 전략 가이드라인
```
1 Go (×2): 거의 항상 이득. 데미지 2배는 매우 큼.
      └→ 예외: 마지막 라운드이고 현재 점수가 충분할 때

2 Go (×3): 현재 배수가 높을 때만.
      └→ 예외: 목숨 1개일 때 (3 Go 리스크 회피)

3 Go (×8): Balatro식 "숫자 폭발" 순간. 목숨을 건 한 방.
      └→ 성공 시 4,000~7,000+ 데미지 가능
      └→ 실패(즉사 10%) 시 게임오버
      └→ 고윤회 보스 돌파의 핵심 전략
```

## 5. 엽전(냥) 경제

### 획득/소비 밸런스

| 행동 | 엽전(냥) 변동 |
|------|-------------|
| 게임 시작 | +100냥 |
| 관문 1~3 보스 격파 | +50~100냥 |
| 관문 4~6 보스 격파 | +100~150냥 |
| 관문 7~9 보스 격파 | +150~200냥 |
| 관문 10 보스 격파 | +300냥 |
| 이벤트 보상 | +20~50냥 |

| 소비 (저승장터) | 엽전(냥) 비용 |
|---------------|-------------|
| 일반 부적 구매 | 30~50냥 |
| 희귀 부적 구매 | 80~120냥 |
| 전설 부적 구매 | 200~300냥 |
| 부적 제거 | 50냥 |
| 카드 팩 (추가 드로우) | 40냥 |
| 체력 회복 | 75냥 |

### 경제 흐름 시뮬레이션
```
시작: 100엽전
1층 클리어 후: 100 + 50 = 150 → 상점: 일반 부적 1개 (-40) = 110
2층 클리어 후: 110 + 75 = 185 → 상점: 희귀 부적 1개 (-100) = 85
3층 클리어 후: 85 + 100 = 185 → 상점: 일반 부적 1개 + 체력 (-40-75) = 70
4층 클리어 후: 70 + 125 = 195 → 상점: 전설 부적 절약 or 소모품
5층 최종전: 남은 엽전으로 마지막 준비
```

**설계 의도:** 매 상점에서 1~2개 구매 가능. 전설 부적은 아껴야 살 수 있음. 긴장감 유지.

## 6. 난이도 곡선

```
난이도 ▲
       │                                    ★ 관문 10
       │                              ★   /
       │                         ★  / \  /
       │                    ★  /  장터  \/
       │               ★  /  격노!    /
       │          ★  / \ /          /
       │     ★  /  장터           /
       │  ★  / (저승장터=쉼)     /
       │★  /                    /
       │  /   보스 항상 랜덤    /
       │ /                     /
       └──────────────────────────→ 진행도
       관문1  관문3  관문5  관문7  관문10
```

### 난이도 완화 요소
- 상점에서 부적 보강
- 이벤트에서 보너스 획득
- Go/Stop으로 플레이어가 난이도 조절
- 부적 시너지로 점수 폭발 가능

### 난이도 강화 요소
- 보스 기믹 점진적 강화
- 보스 HP 증가
- Go 사용 시 다음 라운드 패널티
- 저주 부적 강제 장착 (이벤트)

## 7. 윤회(Cycle) 스케일링

### 무한 윤회 난이도 공식
게임은 10개 관문으로 구성된 윤회가 무한 반복된다. 윤회 회차에 따라 전체 수치가 스케일링된다.

```
윤회 계수 = 1 + (윤회번호 - 1) × 0.3

보스 HP = 관문번호 × 300 × 윤회 계수
보스 기믹 수 = min(기본 기믹 수 + floor(윤회번호 / 3), 최대 4)
보상 배율 = 1 + (윤회번호 - 1) × 0.15
격노 진입 HP% = max(50% - (윤회번호 × 2%), 30%)  // 고윤회에서 빨리 격노
```

### 윤회별 예상 수치
| 윤회 | 관문 1 HP | 관문 10 HP | 기믹 수 | 격노 진입 | 보상 배율 |
|------|----------|-----------|---------|----------|----------|
| 1 | 300 | 3000 | 1 | 50% | x1.0 |
| 2 | 390 | 3900 | 1 | 46% | x1.15 |
| 3 | 480 | 4800 | 1~2 | 44% | x1.30 |
| 5 | 660 | 6600 | 2 | 40% | x1.60 |
| 10 | 1110 | 11100 | 3~4 | 30% | x2.35 |
| 20 | 1860 | 18600 | 4 | 30% | x3.85 |

### 스케일링 설계 원칙
- **윤회 1~3:** 학습 구간. 고스톱 족보 위주로 클리어 가능.
- **윤회 4~7:** 섯다 족보 + 부적 시너지 + Go 활용이 필수. 빌드 최적화 요구.
- **윤회 8~15:** 3종 족보 극대화 + 전설 부적 없이는 불가. 고급 플레이어 영역.
- **윤회 16+:** 극한 도전. 파츠 복합 기믹 + 천문학적 HP → 리더보드 경쟁 영역.

### 보스 파츠 스케일링
- 모든 윤회: 보스 항상 랜덤 (고정 보스 순서 없음)
- 윤회 1~3: 파츠 1~2개 랜덤
- 윤회 4+: 파츠 2~3개 랜덤 조합, 희귀 파츠 등장
- 윤회 8+: 전설 파츠 등장, 복합 기믹 3~4개 동시 적용, 격노 진입 빨라짐
