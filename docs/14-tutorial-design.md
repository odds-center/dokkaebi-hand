# 튜토리얼 설계 문서

## 1. 설계 철학

### 목표
- 화투를 **전혀 모르는 글로벌 유저**도 3분 내에 기본 플레이 가능
- "배운다"는 느낌 없이 **자연스럽게 플레이하면서 학습**
- 핵심 재미(매칭 → 족보 → Go/Stop)를 **순서대로 맛보기**

### 원칙
1. **Show, Don't Tell:** 텍스트 벽 대신 실제 플레이로 가르침
2. **한 번에 하나:** 한 스텝에서 하나의 개념만 소개
3. **실패해도 괜찮게:** 튜토리얼 중 패배 없음, 자유롭게 실험
4. **스킵 가능:** 경험자를 위한 "건너뛰기" 항시 제공

## 2. 튜토리얼 구조 (4단계)

### Step 1: "같은 월의 패를 맞춰라" (60초)

**상황 설정**
```
[삼도천 강가. 뱃사공이 화투패를 내려놓는다]
뱃사공: "화투를 칠 줄 모른다고? 간단해. 보여줄게."
```

**진행**
- 손패 3장, 바닥패 3장만 제공 (단순화)
- 같은 월 카드가 빛남 (시각적 힌트)
- 플레이어가 1월 패를 탭 → 바닥의 1월 패 하이라이트
- "같은 숫자(월)의 패를 맞추면 가져올 수 있어."

**학습 목표**
- [x] 같은 월 매칭 개념
- [x] 탭하여 카드 내기
- [x] 카드 획득

**제약**
- 매칭 실패 불가 (모든 손패에 대응하는 바닥패 배치)
- 뒤집기 단계 생략

---

### Step 2: "족보를 완성해 보스 HP를 깎아라" (60초)

**상황 설정**
```
뱃사공: "카드를 모으는 게 끝이 아니야. 족보를 만들어야 해."
[획득한 카드 옆에 족보 가이드가 반투명하게 표시됨]
```

**진행**
- 손패 5장, 바닥패 6장 제공
- 홍단 완성 가능하도록 배치 (1,2,3월 홍단이 확보 가능)
- 홍단을 모으면 족보 완성 연출
- "홍단 완성! 칩 120 × 배수 3 = 360 데미지!"
- 칩×배수 = 데미지 개념을 시각적으로 설명
- "이 데미지로 보스 HP를 깎는 거야."

**학습 목표**
- [x] 족보 개념 (같은 종류 모으기)
- [x] 칩 × 배수 = 보스에게 주는 데미지
- [x] 족보 가이드 패널 사용법

**제약**
- 뒤집기 매칭은 자동 진행
- 보스 없음 (자유 플레이)

---

### Step 3: "부적으로 데미지를 폭발시켜라" (45초)

**상황 설정**
```
뱃사공: "이건 부적이야. 죽은 자들이 남긴 힘이지."
[피의 맹세 부적이 슬롯에 장착됨]
뱃사공: "피 패를 모을수록 배수가 올라간다."
```

**진행**
- 이전 스텝의 상태 이어서 진행
- 피의 맹세 자동 장착
- 피 패를 5장 모으도록 유도
- 부적 발동 → "+5 배수!" 연출
- "부적은 저승 장터에서 더 살 수 있어. 조합이 핵심이야."

**학습 목표**
- [x] 부적 개념 (패시브/트리거 효과)
- [x] 부적이 데미지에 미치는 영향
- [x] 부적 슬롯 UI 위치

---

### Step 4: "Go or Stop — 욕심이 널 죽인다" (45초)

**상황 설정**
```
뱃사공: "마지막으로 가장 중요한 것. Go와 Stop."
[족보가 완성된 상태에서 Go/Stop 선택지 등장]
뱃사공: "Go를 외치면 배수가 올라가. 하지만..."
```

**진행**
- 족보 완성 상태에서 시작
- Go/Stop 패널 등장
- **1차:** Go를 선택하도록 유도 → 배수 ×2 연출
- **2차:** 다시 족보 달성 → Stop 선택하도록 유도
- "Go는 배수를 올리지만, 3번 Go에 실패하면 즉사야."
- "욕심을 부릴지, 안전하게 갈지는 네가 정해."

**학습 목표**
- [x] Go = 계속 (배수 증가 + 리스크)
- [x] Stop = 확정 (안전하게 데미지 확보 → 보스 HP 타격)
- [x] Go 실패의 위험성

## 3. 튜토리얼 후 첫 실전

### 첫 보스전 도입
```
[튜토리얼 완료]
뱃사공: "기본은 알겠지? 그럼 가자."
뱃사공: "첫 번째 도깨비가 기다리고 있다."
뱃사공: "녀석의 HP를 족보로 깎아서 0으로 만들면 된다."

[먹보 도깨비 등장]
먹보: "크하하! 새로운 밥이 왔구나!"

보스 HP: 150 (튜토리얼 직후라 낮게 설정)
핸디캡: 먹보 기믹 미적용 (첫 라운드만)
```

### 점진적 난이도 해제
| 실전 라운드 | 해제되는 요소 |
|------------|-------------|
| 1라운드 | 기본 매칭 + 뒤집기 (보스 기믹 없음) |
| 2라운드 | 보스 기믹 적용 시작 |
| 3라운드 | 풀 루프 (모든 시스템 활성) |

## 4. 인게임 도움말

### 항시 접근 가능한 가이드
- 화면 우상단 **[?] 버튼** → 족보 가이드 패널 (오버레이)
- 카드 **길게 터치** → 카드 상세 정보 팝업
- 부적 **길게 터치** → 부적 효과 설명 팝업
- 보스 이름 **탭** → 보스 기믹 설명

### 툴팁 시스템
```
[상황별 자동 툴팁]

첫 3장 매칭 → "쓸! 바닥의 같은 월 3장을 전부 가져옵니다."
첫 Go 선택 → "Go: 배수가 2배! 하지만 다음 라운드가 어려워집니다."
첫 부적 발동 → "피의 맹세 발동! 피 패가 많을수록 강해집니다."
첫 보스 기믹 → "먹보 도깨비가 카드를 먹었습니다! 고가치 패를 빨리 사용하세요."
```

### 툴팁 빈도 감소
- 같은 툴팁은 3회까지만 표시
- "툴팁 끄기" 옵션 제공
- 숙련도에 따라 자동 감소

---

## 5. 재방문 유저 대응

### "튜토리얼 건너뛰기"
- 타이틀 화면에서 "새 게임" 선택 시
- "튜토리얼을 진행하시겠습니까?" → [예] / [건너뛰기]
- 건너뛰기 시 피의 맹세 부적 기본 지급 (동일 조건)

### 연습 모드 (Alpha에서 추가)
- 메인 메뉴 → "연습" → 자유 플레이
- 보스 없이 무한 라운드
- 모든 부적 사용 가능 (실험용)
- 족보 가이드 항시 표시
