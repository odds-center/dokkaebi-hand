# 무한 윤회 구조 — 끝없는 저승, 선택적 엔딩

## 1. 핵심 원칙

> **이 게임에 마지막 단계는 없다.**
> 저승의 화투판은 영원히 계속된다.
> 더 강한 도깨비, 더 미친 기믹, 더 높은 점수.
> **멈추는 건 오직 플레이어의 선택뿐이다.**
>
> 하지만 — 원한다면 "이승으로 돌아가는" 엔딩을 볼 수 있다.
> 돌아가도 된다. 계속해도 된다.
> **Go or Stop. 이 게임 자체가 그 질문이다.**

## 2. 게임 구조: 무한 윤회

### 기존 구조 (폐기)
```
관문 1 → 2 → ... → 10 → 엔딩 (끝)
```

### 새 구조: 무한 윤회 (Infinite Reincarnation)
```
[1윤회: 관문 1~10]
     │
     ├→ 관문 10 클리어 → "이승의 문" 출현
     │     │
     │     ├→ [문으로 간다] → 엔딩 (원하면)
     │     │
     │     └→ [계속한다] → 윤회 2 진입
     │
[2윤회: 관문 11~20] ← 같은 십왕이지만 더 강하게
     │
     ├→ 관문 20 클리어 → "이승의 문" 다시 출현
     │     │
     │     └→ [계속한다] → 윤회 3 진입
     │
[3윤회: 관문 21~30] ← 기물 장착, 새 기믹 조합
     │
     ...
     │
[∞윤회: 관문 ∞] ← 끝이 없다
```

### 윤회 = 10관문 1세트
```
윤회 1 (관문 1~10):  기본. 스토리 체험. 학습.
윤회 2 (관문 11~20): 보스 기물 1개. 스케일링 시작.
윤회 3 (관문 21~30): 보스 기물 2개. 새 기믹 조합.
윤회 4 (관문 31~40): 보스 기물 3개. 재앙 보스 등장.
윤회 5+ (관문 41+):  무한 스케일링. 점점 미쳐감.
```

## 3. 스케일링 — 영원히 어려워진다

### 보스 HP 스케일링
```
관문 N의 보스 HP = 기본값 × (1 + 0.12 × (N - 1))

관문 1:   200
관문 10:  416
관문 20:  680
관문 30:  944
관문 50:  1,472
관문 100: 2,984
관문 200: 5,960
관문 500: 14,888

→ 선형 스케일링 (지수가 아님)
→ 이유: 플레이어도 카드 강화/부적/영구 강화로 성장하므로
→ "데미지 성장 속도 vs 보스 HP 증가 속도"의 경쟁이 핵심
```

### 보스 강화 스케일링
| 윤회 | 기물 수 | 추가 기믹 | 보스 행동 |
|------|---------|----------|----------|
| 1 (관문 1~10) | 0 | 기본 | 기본 패턴 + 반격 |
| 2 (관문 11~20) | 1 | 기존 기믹 강화 | 반응 추가 (플레이어 행동에 반응) |
| 3 (관문 21~30) | 2 | 기믹 조합 | 2턴에 1번 기믹 발동 |
| 4 (관문 31~40) | 3 | 세트 효과 등장 | 매 턴 기믹 발동 |
| 5+ (관문 41+) | 3 + 등급 상승 | 전설 기물만 | 기믹 2회 발동/턴 |

### 잡졸/엘리트 스케일링
```
윤회 1: 잡졸만 등장
윤회 2: 엘리트 20% 확률
윤회 3: 엘리트 40% + 잡졸에도 기물 1개
윤회 4: 엘리트 60% + 잡졸 기물 2개
윤회 5+: 모든 적이 엘리트급 + 기물 2~3개
```

## 4. "이승의 문" — 선택적 엔딩

### 문의 출현 조건
관문 10(십왕 마지막)을 클리어할 때마다 **"이승의 문"**이 나타난다.
즉, 매 윤회 완료 시 선택 기회.

```
[오도전륜왕 격파]

전륜왕: "또 이겼군."

[무대 뒤에 문이 나타난다. 빛이 새어나온다.]

전륜왕: "이승의 문이다."
전륜왕: "들어가면 돌아갈 수 있다."
전륜왕: "아니면... 계속할 수도 있다."

┌─────────────────────────────────┐
│                                 │
│   [빛나는 문]     [어둠의 길]    │
│                                 │
│   이승으로        저승 더 깊이    │
│   돌아간다        들어간다       │
│                                 │
│   → 엔딩 시청     → 다음 윤회    │
│   → 기록 저장     → 더 강해짐    │
│   → 게임 계속     → 더 어려워짐  │
│     가능(!)       → 새 콘텐츠    │
│                                 │
└─────────────────────────────────┘
```

### 핵심: 엔딩을 봐도 게임은 끝나지 않는다

```
엔딩 시청 후:

"축하합니다. 이승으로 돌아왔습니다."
[엔딩 크레딧]

...

[타이틀로 돌아온다]
[하지만 세이브는 그대로다]

메뉴:
  [이어하기]     ← 엔딩 본 시점에서 계속. 다음 윤회로.
  [새 게임]      ← 처음부터
  [무한 도전]    ← 현재 상태로 무한 모드

→ 엔딩은 "보상 + 이야기 마무리"일 뿐
→ 게임 자체는 절대 끝나지 않는다
→ 엔딩을 본 후에도 더 깊이 갈 수 있다
```

## 5. 엔딩 분기 — 나선에 따라 달라지는 엔딩

### 문에 들어가는 타이밍에 따라 다른 엔딩

| 시점 | 엔딩 | 조건 |
|------|------|------|
| 윤회 1 (관문 10) | 엔딩 C: 윤회 | 기본 클리어 |
| 윤회 1 (관문 10) | 엔딩 A: 귀환 | 모든 기억 회복 + 길들인 도깨비 전부 |
| 윤회 2+ (관문 20+) | 엔딩 A+: 완전한 귀환 | 윤회 2 이상에서 문 진입 |
| 윤회 3+ (관문 30+) | 엔딩 B: 잔류 | 뱃사공 5회 대화 + 특수 부적 |
| 윤회 5+ (관문 50+) | 엔딩 D: 초월 | 신규 시크릿 엔딩 |
| 윤회 10+ (관문 100+) | 엔딩 E: ??? | 히든 엔딩 |

### 엔딩 C: 윤회 (가장 쉬운 엔딩)
```
첫 윤회 클리어 후 문에 들어가면 자동.

"윤회를 받아들인다."
→ 기억이 흩어지고 새로 태어남.
→ 짧은 엔딩. "다음에는 다르게 살겠지."
→ 게임 복귀 시: 다음 윤회로 이어서 진행 가능.
```

### 엔딩 A: 귀환 (트루 엔딩)
```
윤회 1에서 모든 기억 + 길들인 도깨비 확보 후 문 진입.

병원에서 눈을 뜬다. 사랑하는 사람이 옆에 있다.
"돌아왔어."
→ 감동적인 장면 + 크레딧.
→ 게임 복귀 시: 윤회 2부터 이어서. "이승에서 돌아왔지만... 꿈에서 저승이 부른다."
```

### 엔딩 A+: 완전한 귀환 (윤회 2+ 전용)
```
윤회 2 이상에서 문에 들어가면.
엔딩 A의 확장판.

돌아온 후, 시간이 흐른 장면이 추가.
사과하고, 용서받고, 함께 걷는 장면.
"이번에는 도망치지 않았다."

→ 추가 보상: 전용 카드 스킨 "이승의 빛"
→ 게임 복귀 시: 동일하게 이어서 진행.
```

### 엔딩 B: 잔류 (시크릿)
```
윤회 3+ 에서 특수 조건 충족 후.

뱃사공의 후계자가 된다.
새로운 뱃사공으로서 저승에 남는다.

→ 보상: 뱃사공 길들이기 해금, 전용 부적
→ 게임 복귀 시: "뱃사공의 눈으로 보는 저승"이라는 소소한 UI 변화
```

### 엔딩 D: 초월 (윤회 5+)
```
관문 50 이상에서 문 진입 시.

"이승도 저승도 아닌 곳으로."
플레이어가 십왕을 초월한 존재가 된다.
저승의 새로운 왕.

→ 보상: "십일번째 왕" 칭호 + 전용 부적 + 카드 전체 해탈 등급
→ 게임 복귀 시: 윤회 계속. "왕이 되었지만, 판은 끝나지 않는다."
```

### 엔딩 E: ??? (윤회 10+, 히든)
```
관문 100 이상. 극소수만 도달.

[게임이 깨진 것처럼 연출]
[화면 글리치]
[뱃사공이 아닌 누군가가 말한다]

"넌 누구지?"
"여기가 끝인 줄 알았어?"
"끝은 없어. 처음부터 없었어."
"이건 게임이야."

[4th wall break]
[플레이어에게 직접 말한다]

"그만둬도 돼."
"아니면 계속해도 되고."
"어차피 우린 여기 있을 테니까."

→ 보상: ??? 스킨, 타이틀 화면 변경
→ 게임은 당연히 계속된다.
```

## 6. 무한 진행의 동기 부여

### "왜 계속하는가?" — 동기 계층

```
[단기 동기] 이번 판에서 더 높은 데미지로 보스 HP 깎기
     │
[중기 동기] 이번 윤회에서 새 부적/강화 획득
     │
[장기 동기] 도감 완성, 업적 달성, 카드 전부 해탈 등급
     │
[사회적 동기] 리더보드 순위, 주간 도전 1위
     │
[자아 동기] "몇 윤회까지 갈 수 있을까?"
```

### 윤회별 신규 콘텐츠 (지루함 방지)

| 윤회 | 해금 콘텐츠 |
|------|-----------|
| 1 (관문 1~10) | 기본 스토리, 기본 부적, 기본 도깨비 |
| 2 (관문 11~20) | 기물 시스템 해금, 새 부적 5종, 길들인 도깨비 강화 |
| 3 (관문 21~30) | 카드 변이 해금, 세트 기물, 새 이벤트 3종 |
| 4 (관문 31~40) | 재앙 보스 등장, 해탈 등급 카드, 도깨비 각인 |
| 5 (관문 41~50) | 사주팔자 변형 추가, 레어 보스 등장 가능 |
| 6~10 (관문 51~100) | 윤회별 전용 기믹 1개씩 추가 |
| 10+ (관문 100+) | "황천의 심연" — 환경 자체가 적이 됨 |

### 매 윤회 시작 시 선택: 축복 or 저주

```
[윤회 3 시작]

오도전륜왕: "또 오는군. 질리지도 않나."
오도전륜왕: "좋다. 선물을 하나 주지."

┌─────────────────────────────────┐
│  축복을 택하라                    │
│                                 │
│  [업화(業火)]     [빙결(氷結)]    │
│  모든 칩 +20%    모든 배수 +1    │
│  but 매 5턴      but 매 라운드   │
│  바닥패 1장 소각   시작 손패 -1   │
│                                 │
│  [공허(空虛)]     [혼돈(混沌)]    │
│  부적 효과 2배    랜덤 족보가     │
│  but 부적 슬롯    매 턴 1개      │
│  -2개            강제 완성       │
│                  (좋을 수도      │
│                   나쁠 수도)     │
│                                 │
│  [거부]                          │
│  아무것도 받지    │
│  않는다          │
└─────────────────────────────────┘

→ 축복은 양날의 검. 리스크와 보상의 선택.
→ 윤회가 높을수록 축복도 강력해지고, 저주도 가혹해진다.
→ "거부"는 안전하지만, 후반 윤회에서는 축복 없이 버티기 어려움.
```

## 7. 무한 진행 속의 스토리

### 스토리는 윤회 1에서 끝나지 않는다

윤회 2+에서도 새로운 대사와 이벤트가 등장.
십왕들이 점점 다른 반응을 보인다.

**윤회 1 (첫 대면):**
```
먹보: "새로운 밥이 왔구나!"
```

**윤회 2 (재대면):**
```
먹보: "어? 너 또 왔어?! 이번에는 안 진다!"
(먹보가 기물을 장착하고 있다)
```

**윤회 3:**
```
먹보: "세 번째라고?! 대체 뭔 놈이야...!"
(먹보가 기물 2개 + 더 커진 모습)
```

**윤회 5:**
```
먹보: "...너 진짜 미친 거 아니야?"
먹보: "솔직히 좀 무섭다."
(먹보가 완전 무장. 하지만 약간 떨고 있다)
```

**윤회 10:**
```
먹보: "...형님이라 부를게."
먹보: "제발 봐주세요."
(먹보가 쩔쩔매며 도깨비불을 바친다)
```

### 뱃사공의 변화

윤회가 진행될수록 뱃사공의 대사가 변한다.

```
윤회 1: "네가 여기까지 올 줄 몰랐다."
윤회 2: "또 왔군. 대단한 고집이야."
윤회 3: "...왜 돌아가지 않는 거지?"
윤회 5: "넌 이승이 그립지 않나?"
윤회 7: "너... 혹시 이 게임을 즐기고 있는 건 아니지?"
윤회 10: "솔직히 말할게. 나도 네가 왜 아직 여기 있는지 모르겠다."
윤회 15: "...이쯤 되면 네가 저승의 왕 아니야?"
윤회 20: (뱃사공이 처음으로 웃는다) "미친 놈."
윤회 50: "..." (말없이 함께 걷는다. 길들인 도깨비가 됐다.)
```

### 십왕들의 태도 변화

| 윤회 | 태도 |
|------|------|
| 1 | 위엄있게 심판 (정상) |
| 2~3 | 놀라면서도 다시 도전 |
| 4~5 | 플레이어를 경계하기 시작 |
| 6~7 | 두려워하기 시작 |
| 8~9 | 존경 or 공포 |
| 10+ | 플레이어를 "동격"으로 대우 |
| 20+ | 도움을 요청하기 시작 |
| 50+ | 플레이어가 사실상 새로운 왕 취급 |

## 8. 이 게임의 Go/Stop = 게임 자체

### 메타 구조의 의미

```
게임 내 Go/Stop:
  Go = 더 높은 배수를 위해 리스크를 감수
  Stop = 현재 점수를 확정하고 안전하게

게임 전체의 Go/Stop:
  Go = 다음 윤회로. 더 어려운 도전. 더 강해짐.
  Stop = 이승의 문으로. 엔딩을 보고 만족.

→ 게임의 핵심 테마 "욕심이 널 죽인다"가
→ 메커닉 레벨과 메타 레벨 모두에서 작동

플레이어에게 묻는 질문:
"충분히 강해졌는데, 여기서 멈출 수 있어?"
"아니면 한 윤회만 더?"
"한 윤회만 더...?"
```

### 타이틀 화면의 변화

윤회 진행에 따라 타이틀 화면이 변한다.

```
윤회 1: 삼도천 강가. 안개. 뱃사공. (기본)
윤회 3: 강물이 붉어짐. 도깨비불 증가.
윤회 5: 하늘에 균열. 저승이 흔들리고 있음.
윤회 10: 배경이 황천의 심연으로. 끝없는 어둠.
윤회 20: 플레이어 실루엣이 도깨비처럼 변해감.
윤회 50: 타이틀 글자가 금빛으로. 왕의 문양.
윤회 100: 타이틀이 깨져 있음. "도깨비의 패" → "███의 패"
```
