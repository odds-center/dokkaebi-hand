# 감정적 경험 설계 — 플레이어가 느끼는 저승의 도박장

> 시스템 룰을 넘어, 플레이어가 게임을 켰을 때 느끼는
> **시각, 청각, 그리고 감정적인 경험**의 총체적 설계.

---

## 1. 시각적 연출: 먹물과 도깨비불이 섞인 저승

### 무대 — 어둠 속의 도박장

화면 전체는 짙은 안개가 깔린 어두운 방이다.
플레이어 앞에는 낡은 **붉은 모포**가 깔려 있고,
건너편 어둠 속에서는 도깨비의 **번뜩이는 눈동자**만 보인다.

```
[화면 레이어 구성]

Layer 5 (최상단): 파티클 — 도깨비불, 안개, 먹물 파편
Layer 4: UI — 점수, 배수, 목표, 부적 슬롯
Layer 3: 카드 — 손패, 바닥패 (인터랙션 영역)
Layer 2: 테이블 — 붉은 모포, 먹물 자국, 엽전 더미
Layer 1: 배경 — 어둠, 안개, 도깨비 실루엣
Layer 0: 앰비언트 — 미세한 빛 변화, 분위기 조명
```

### 화투패의 질감

일반적인 플라스틱 화투가 아니다.
**오래된 한지(韓紙)**를 깎아 만든 듯한 질감.
패에 그려진 붉은 태양(광)이나 벚꽃은
마우스를 올릴 때마다 **도깨비불 색상으로 은은하게 빛난다**.

```
[카드 호버 상태]

기본:    한지 질감, 먹물 그림, 차분한 색조
호버:    테두리에 파란 도깨비불 미세하게 발광
         카드가 살짝 떠오르는 느낌 (Y축 +3px, 그림자 확장)
         카드 표면에 미세한 먹물 물결 셰이더
선택:    도깨비불 강하게 발광 + 카드 확대
         주변 카드 어둡게 (포커스 효과)
```

### 시너지 발동 — 피가 광이 되는 순간

> "피의 맹세" 부적이 발동되면,
> 획득한 피 카드들에서 **진짜 붉은 핏줄**이 솟아올라
> 화면 중앙으로 모인 뒤, 눈부신 **'光(광)' 글자로 폭발**하며
> 점수가 치솟는다.

```
[부적 발동 연출 시퀀스 — 피의 맹세]

0.0초: 부적 아이콘 빛남 (도깨비불 초록)
0.2초: 획득한 피 카드들이 각각 붉게 발광 시작
0.5초: 각 피 카드에서 붉은 먹물 줄기가 뻗어나옴
0.8초: 먹물 줄기들이 화면 중앙으로 수렴
1.0초: 중앙에서 폭발 — 먹물이 원형으로 퍼짐
1.2초: 배수 숫자가 빠르게 카운트업 (+1, +2, +3...)
1.5초: "피의 맹세!" 서예체 텍스트 + 최종 배수 확정
2.0초: 여운 — 붉은 파티클 서서히 사라짐
```

### 보스의 존재감

보스는 화면 상단 어둠 속에 **실루엣으로** 존재.
평소에는 눈동자만 빛나지만,
기믹 발동 시 어둠에서 **몸체가 드러나며** 위협.

```
[보스 존재감 레벨]

Lv 1 (평온): 눈동자만 빛남. 가끔 깜빡임.
Lv 2 (관심): 플레이어가 좋은 패를 내면 눈이 커짐.
Lv 3 (위협): 기믹 발동 시 어둠에서 손/팔이 나타남.
Lv 4 (분노): 플레이어 점수가 높으면 테이블을 내려침 (화면 진동).
Lv 5 (공포): 3 Go 선택 시 보스 전체 모습이 드러남 + 포효.
```

## 2. 청각적 쾌감: 영혼을 때리는 찰진 손맛

### 타격감 (손맛) — 화투의 "착!"

화투의 핵심은 패를 내려칠 때 나는 **"착!"** 소리.
마우스로 패를 낼 때마다 **묵직한 돌덩이를 내려찍는 듯한 베이스 사운드**와 함께
화면이 미세하게 흔들린다(Screen Shake).

```
[타격 사운드 레이어링]

Layer 1: 기본 타격 — "탁!" (나무 위에 패를 놓는 소리)
Layer 2: 서브 베이스 — "쿵" (저음역 진동, 80Hz)
Layer 3: 상황별 레이어:
         매칭 성공: + 금속 공명 "챙~"
         쓸:        + 바람 "쏴아~" + 종이 쓸리는 소리
         매칭 실패: + 먹물 떨어지는 "뚝"

[콤보 에스컬레이션]
1연속 매칭: 기본 타격
2연속:      타격 + 서브베이스 볼륨 ↑
3연속:      타격 + 메탈릭 파열음 추가
4연속+:     타격 + 화면 흔들림 강화 + 잔향 증가
```

### 배경음악 (BGM) — 국악 × 다크 일렉트로닉

```
[평상시]
BPM: 85
구성: 가야금 아르페지오 + 해금 롱톤 + 앰비언트 패드
분위기: 느리고 으스스. 저승의 정적.
드럼: 없음. 가끔 장구 한 박.

[Go 1회 선택]
BPM: 100 (가속)
추가: 장구 리듬 시작 + 신스 베이스라인
분위기: 긴장감 시작. 심장이 뛰기 시작.

[Go 2회 선택]
BPM: 120 (가속)
추가: 808 킥 + 하이햇 + 가야금 트레몰로
분위기: 거의 전투 수준. 돌이킬 수 없는 느낌.

[Go 3회 선택!!!]
BPM: 140 (최대)
추가: 풀 드럼 + 디스토션 거문고 + 사이렌 신스
분위기: 미친 도박. 심장이 터질 것 같은 긴장.
→ 갑자기 모든 소리 멈춤 (0.5초 정적)
→ 결과에 따라:
   성공: 폭발적 팡파레 + 금관 + 환호
   실패: 단일 장구 "쿵..." + 정적 + 균열음
```

### 사운드 디테일

| 순간 | 소리 | 감정 |
|------|------|------|
| 패를 집을 때 | 종이 스치는 "스읍" | 기대감 |
| 패를 놓을 때 | 묵직한 "착!" + 미세 진동 | 결단력 |
| 매칭 성공 | 맑은 종 "챙~" + 바람 | 쾌감 |
| 매칭 실패 | 먹물 "뚝..." + 정적 | 아쉬움 |
| 쓸 (3장 매칭) | 강풍 "쏴아!" + 종이 쓸림 + 확대 | 통쾌함 |
| 족보 완성 | 북 "둥!" + 징 "쨍!" + 합창 | 성취감 |
| Go 선택 | 심장 박동 + 도깨비 "크크크..." | 공포+흥분 |
| Stop 선택 | 징 한 번 "쨍!" + 안도의 한숨(앰비언트) | 안도감 |
| 보스 기믹 | 우레 + 도깨비 으르렁 + 화면 진동 | 위기감 |
| 보스 패배 | 폭발 + 도깨비 비명 + 사슬 끊는 소리 | 정복감 |
| 점수 카운트업 | 주판 "딸깍딸깍" 가속 → 최종 "쿵!" | 기대→만족 |

## 3. 내러티브 분위기: 포식자와의 도박

### 도깨비는 딜러이자 포식자

도깨비들은 단순한 AI가 아니다. **살아있는 딜러**다.
플레이어의 행동에 **실시간으로 반응**한다.

```
[도깨비 반응 시스템]

▸ 플레이어가 오래 고민할 때:
  → 도깨비가 손가락으로 테이블을 "똑똑" 두드림
  → "어서 내놔. 밤새 할 거야?" (텍스트)
  → 어둠 속에서 짜증난 숨소리

▸ 플레이어가 실수했을 때:
  → 도깨비 눈이 초승달 모양으로 웃음
  → 낮은 "크크크..." 웃음소리
  → "역시 이승 것들은 어리석어." (텍스트)

▸ 플레이어가 좋은 족보를 완성했을 때:
  → 도깨비 눈동자가 커짐 (놀람)
  → 테이블 "쿵" 내려침 + 화면 흔들림
  → "뭐?! 이 녀석이...!" (텍스트)

▸ 플레이어가 3 Go를 선택했을 때:
  → 도깨비가 어둠에서 완전히 모습을 드러냄
  → 입이 귀까지 찢어진 미소
  → "좋아... 좋아... 마음에 드는군." (텍스트)
  → 도깨비불이 미친듯이 타오름
```

### 정복감 — 보스를 압도하는 순간

플레이어의 점수가 보스의 목표를 **크게 초과**하기 시작하면:

```
[보스 동요 연출]

점수 50% 달성: 도깨비 눈빛 흔들림 시작
점수 80% 달성: 도깨비가 의자에서 일어남 (불안)
점수 100% 달성: 도깨비 얼굴에 균열 이펙트
점수 200%+ 달성: 도깨비가 뒤로 밀려남 + 테이블에 금이 감
                  "말도 안 돼...!" 텍스트가 떨림
점수 500%+ 달성: 도깨비 실루엣이 산산조각
                  화면 전체 백색 플래시
                  "기억하마... 네 이름을..." 마지막 대사
```

## 4. 절정의 순간: "쓰리 고(3 Go)" 시퀀스

> 안전하게 멈출 수 있었지만, 도파민에 중독된 플레이어가
> 붉게 물든 **[GO]** 버튼을 세 번째 누르는 순간.

### 연출 시퀀스 (7초)

```
[0.0초] — 선택의 순간
  화면 전체 색감이 채도 0으로 (흑백에 가까운)
  [GO] 버튼만 핏빛으로 맥박처럼 깜빡
  배경음악 완전 정지
  심장 박동 소리만 "쿵... 쿵... 쿵..."

[0.5초] — 버튼 클릭
  버튼에서 핏빛 충격파 발생
  화면 전체 색감이 핏빛으로 반전
  "쿵!!" 단일 임팩트 사운드 (서브우퍼 급)
  화면 2프레임 정지 (히트스톱)

[1.0초] — 정적
  완전한 정적 0.5초
  화면 어두워짐
  도깨비 눈동자만 빛남 → 입꼬리가 올라감

[1.5초] — 폭발
  BGM 폭발적으로 재개 (BPM 140, 풀 오케스트라)
  화면에 먹물 폭풍 이펙트
  배수 카운터 슬롯머신처럼 회전 시작
  "x10... x50... x120... x450... x1200!"

[3.0초] — 점수 확정 (성공 시)
  배수 최종 확정 "x1,847!"
  금빛 폭발 + 화면 가장자리에서 빛줄기
  도깨비가 테이블에서 날아감 (과장된 연출)
  점수 숫자 거대하게 표시: "247,590!"
  "大勝!" 서예체 텍스트 + 먹물 퍼짐

[5.0초] — 여운
  금빛 파티클이 서서히 내려앉음
  BGM이 승리 멜로디로 전환
  전설 부적 획득 팝업
  도깨비 패배 대사 (떨리는 목소리로)

[7.0초] — 복귀
  화면 정상 색감으로 복귀
  다음 라운드 준비
```

### 3 Go 실패 시

```
[0.0~1.5초] — 동일

[1.5초] — 정적이 길어짐
  정적 1.0초 (성공보다 길다)
  도깨비 웃음소리가 작게 시작 "크크크..."

[2.5초] — 붕괴
  화면에 균열 이펙트 (유리 깨지듯)
  손패가 하나씩 먹물 속으로 빨려들어감
  배수 카운터: "x10... x50... → 0"
  "욕심이 널 삼켰다..." 텍스트 (먹물로)

[4.0초] — 암전
  도깨비 포효 + 테이블 뒤집힘 (사운드)
  화면 완전 암전
  심장 박동 정지음 "────"

[5.0초] — 사망
  "목숨이 사라졌습니다"
  어둠 속에서 도깨비 대사:
  "재밌었다. 또 놀자꾸나."
```

## 7. 촉각적 피드백 (진동/햅틱)

### Steam Deck / 게임패드 햅틱

| 순간 | 진동 패턴 |
|------|----------|
| 패 내려놓기 | 짧고 강한 "탁" (50ms, 강도 0.8) |
| 매칭 성공 | "탁" + 미세 여운 (100ms, 감쇄) |
| 쓸 | 스윕 진동 (좌→우, 200ms) |
| 족보 완성 | 3회 연속 펄스 (둥! 둥! 둥!) |
| Go 선택 | 심장박동 패턴 (쿵-쿵... 쿵-쿵... 가속) |
| 3 Go 폭발 | 최대 강도 연속 진동 (500ms) → 정지 → 폭발 |
| 보스 기믹 | 저주파 연속 진동 (으르르르) |
| 사망 | 강한 한 발 → 정적 |
