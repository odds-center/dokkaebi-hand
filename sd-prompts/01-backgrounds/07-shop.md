# 배경 07 — 상점

## 출력 크기
- **최종 크기: 1920x1080px (전체 화면 배경)**
- 사용 화면: 저승 장터 (상점) 화면
- UI 오버레이: 상점 패널이 화면 대부분을 덮음, 버튼(300x55) 여러 개 배치

## 장면 설명
귀시장의 한 가판대. 물건이 진열된 카운터. 등불.
**느낌: 잠시 쉬어가는 안식처. 따뜻하지만 어딘가 수상하다.**

## 색감 지시
- 주조: 먹빛 (#1A1A2E) — 배경 어둠
- 강조: 따뜻한 주황/금색 — 등불, 물건 반짝임
- 보조: 나무 갈색 — 가판대, 선반
- 포인트: 도깨비불 청색 (#00D4FF) — 특별 물건의 빛

## 사용 팁
- 따뜻함과 아늑함이 핵심 — 다른 배경들과 대비되는 안식처
- 물건 디테일은 과하지 않게 — 반짝임과 색감으로 충분
- 상점은 UI가 위에 많이 올라가므로 **중앙 하단은 비워두는 구도**가 좋음
