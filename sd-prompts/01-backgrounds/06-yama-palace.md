# 배경 06 — 5영역: 염라전 (閻羅殿) / 심판의 궁전

## 출력 크기
- **최종 크기: 1920x1080px (전체 화면 배경)**
- 사용 화면: 5영역 전투 배경 (최종 보스 영역)
- UI 오버레이: 하단 손패 영역(950x160), 상단 보스 HP 바(600x30), 점수 패널(400x80)

## 장면 설명
거대한 심판의 궁전. 금빛 기둥, 옥좌, 벽화. 위에서 쏟아지는 빛.
**느낌: 압도적 위엄. 숨이 멎는 장엄함. 이곳에서 모든 것이 결정된다.**

## 색감 지시
- 주조: 검정/짙은 보라 — 궁전 그림자
- 강조: 금색 (#FFD700) — 기둥, 빛, 장식
- 보조: 핏빛 (#C41E3A) — 벽화, 장식 끈
- 포인트: 먹빛 (#1A1A2E) — 천장, 깊은 그림자

## 사용 팁
- 금빛이 핵심 — 어둠 속에서 금색이 강렬하게 빛나야 함
- 기둥의 반복 패턴은 웅장함을 잘 표현함
- 빛줄기(light beam)는 이미지 생성 AI의 강점 중 하나
- Guidance 3.5~4.0 — 궁전의 구조가 뚜렷해야 함
