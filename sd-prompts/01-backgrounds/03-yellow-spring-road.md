# 배경 03 — 2영역: 황천길 (黃泉路)

## 장면 설명
끝이 보이지 않는 안개 속 길. 오래된 이정표. 도깨비 발자국 흔적.
**느낌: 불안하고 외롭다. 길을 잃었다. 돌아갈 수도 없다.**

## 색감 지시
- 주조: 청회색 — 안개, 하늘
- 강조: 희미한 백색 — 빛줄기
- 보조: 먹빛 (#1A1A2E) — 어둠 깊은 곳
- 포인트: 희미한 초록 — 도깨비 흔적

---

```

(korean underworld yellow spring path:1.2),
(dense volumetric fog:1.3) obscuring everything beyond 20 meters,
old weathered wooden signposts along the road with faded text,

```

## 사용 팁
- 안개가 주인공 — fog 관련 토큰에 가중치를 높게
- 너무 많은 요소를 넣으면 안개 느낌이 사라짐
- CFG 5~6으로 낮추면 더 몽환적인 안개 효과
- 길이 화면 중앙에서 멀리 뻗어가는 구도가 가장 효과적
