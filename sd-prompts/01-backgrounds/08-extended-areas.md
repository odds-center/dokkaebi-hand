# 배경 08~12 — 확장 영역 (6~10영역)

## 출력 크기
- **최종 크기: 1920x1080px (전체 화면 배경)**
- 사용 화면: 6~10영역 전투 배경
- UI 오버레이: 하단 손패 영역(950x160), 상단 보스 HP 바(600x30), 점수 패널(400x80)

## 공통 참고
확장 영역은 나선(Spiral) 진행 시 등장하는 추가 지역.
기본 5영역보다 더 극단적이고 초현실적인 분위기.

## 확장 영역 사용 팁
- 화산(06): 나락(04)의 강화판 — 더 크고, 더 뜨겁고, 더 위험
- 황금 미궁(07): 유일하게 밝은 배경 — 밝음 자체가 함정
- 무한 회랑(08): 미니멀리즘이 핵심 — 요소를 최소화
- 그림자 도시(09): 가장 어두운 배경 — 거의 보이지 않아야 함
- 윤회의 문(10): SD의 "이상한 합성" 성질을 오히려 활용 — 초현실이 맞음
