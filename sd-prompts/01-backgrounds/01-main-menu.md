# 배경 01 — 메인 메뉴: 삼도천 (三途川)

## 출력 크기
- **최종 크기: 1920x1080px (전체 화면 배경)**
- UI 오버레이가 위에 올라가므로 중앙보다 상단/하단에 디테일 집중
- 상단 200px, 하단 150px은 UI 버튼으로 가려짐 — 핵심 비주얼은 중앙~중상단

## 장면 설명
삼도천 강가. 게임의 첫 인상. 안개 낀 강, 낡은 나무배, 떠다니는 도깨비불.
**느낌: 쓸쓸하지만 신비롭다. 무섭지 않고 아련하다.**

## 색감 지시
- 주조: 먹빛 남색 (#1A1A2E) — 하늘, 물
- 강조: 도깨비불 청색 (#00D4FF) — 물 위의 불꽃들
- 보조: 먹물 회색 (#2D2D44) — 산, 안개
- 포인트: 희미한 보라 — 먼 수평선

## 사용 팁
- 3가지 프롬프트를 모두 돌려보고 분위기가 가장 맞는 것 선택
- 메인 메뉴는 게임의 첫인상이므로 **가장 많은 시간을 투자**할 것
- Guidance 3.0으로 낮추면 더 몽환적, 4.0으로 높이면 더 선명
- 도깨비불 개수가 너무 많으면 프롬프트에서 "a few scattered" 등으로 조절
